Funções Comuns¶

O CodeIgniter usa algumas funções para sua operação que são globalmente definidos e estão disponíveis para você a qualquer momento. Estes não exigem carregando quaisquer librarys ou helpers.

is_php($version)¶
Parâmetros:
  • $version (string) – Version number
Returns:

TRUE se a versão em execução do PHP é pelo menos aquela especificada ou FALSE se não for
Return type:

bool

Determines if the PHP version being used is greater than the supplied version number.

Example:

if (is_php('5.3'))
{
        $str = quoted_printable_encode($str);
}

Returns boolean TRUE if the installed version of PHP is equal to or maior que o número da versão fornecido. Retorna FALSE se o arquivo instalado version of PHP is lower than the supplied version number.

is_really_writable($file)¶
Parâmetros:
  • $file (string) - Caminho de arquivo
Returns:

TRUE if the path is writable, FALSE if not
Return type:

bool

is_writable() retorna TRUE em servidores Windows quando você realmente não pode write to the file as the OS reports to PHP as FALSE only if the read-only attribute is marked.

Esta função determina se um arquivo é realmente gravável tentando para escrever primeiro. Geralmente recomendado apenas em plataformas onde this information may be unreliable.

Example:

if (is_really_writable('file.txt'))
{
        echo "I could write to this if I wanted to";
}
else
{
        echo "File is not writable";
}

Nota

See also PHP bug #54709 for more info.

config_item($key)¶
Parâmetros:
  • $key (string) - Chave do item de configuração
Returns:

Configuration key value or NULL if not found
Return type:

mixed

o Config Library is the preferred way of accessing configuration information, however config_item() can be used para recuperar chaves individuais. Vejo Config Library documentation for more information.

set_status_header($code[, $text = ''])¶
Parâmetros:
  • $code (int) – HTTP Response status code
  • $text (string) – A custom message to set with the status code
Return type:

void

Permits you to manually set a server status header. Example:

set_status_header(401);
// Sets the header as:  Unauthorized

See here for a full list of headers.

remove_invisible_characters($str[, $url_encoded = TRUE])¶
Parâmetros:
  • $str (string) – Input string
  • $url_encoded (bool) – Whether to remove URL-encoded characters as well
Returns:

Sanitized string
Return type:

string

Esta função impede a inserção de caracteres NULL entre ASCII caracteres, como Java \ 0script.

Example:

remove_invisible_characters('Java\\0script');
// Returns: 'Javascript'
html_escape($var)¶
Parâmetros:
  • $var (mixed) - Variável para escapar (string ou array)
Returns:

HTML escaped string(s)
Return type:

mixed

Esta função atua como um alias para o nativo do PHP htmlspecialchars() function, with the advantage of being able to accept an array of strings.

It is useful in preventing Cross Site Scripting (XSS).

get_mimes()¶
Returns:An associative array of file types
Return type:array

This function returns a reference to the MIMEs array from application/config/mimes.php.

is_https()¶
Returns:TRUE if currently using HTTP-over-SSL, FALSE if not
Return type:bool

Returns TRUE if a secure (HTTPS) connection is used and FALSE em qualquer outro caso (incluindo solicitações não HTTP).

is_cli()¶
Returns:Verdadeiro se atualmente em execução no CLI, FALSE caso contrário
Return type:bool

Retorna TRUE se o aplicativo for executado por meio da linha de comando e FALSE se não.

Nota

This function checks both if the PHP_SAPI value is ‘cli’ or if the STDIN constant is defined.

function_usable($function_name)¶
Parâmetros:
  • $function_name (string) – Function name
Returns:

TRUE se a função puder ser usada, FALSE se não for
Return type:

bool

Returns TRUE if a function exists and is usable, FALSE otherwise.

This function runs a function_exists() check and if the Suhosin extension <http://www.hardened-php.net/suhosin/> is loaded, checks if it doesn’t disable the function being checked.

É útil se você quiser verificar a disponibilidade de funções such as eval() and exec(), which are dangerous and might be disabled on servers with highly restrictive security policies.

Nota

This function was introduced because Suhosin terminated script execution, but this turned out to be a bug. A fix has been available for some time (version 0.9.34), but is unfortunately not released yet.

Funções Comuns¶

O CodeIgniter usa algumas funções para sua operação que são globalmente definidos e estão disponíveis para você a qualquer momento. Estes não exigem carregando quaisquer librarys ou helpers.

is_php($version)¶
Parâmetros:
  • $version (string) - Número da versão
Retorna:

TRUE se a versão em execução do PHP é pelo menos aquela especificada ou FALSE se não for
Tipo de retorno:

bool

Determina se a versão do PHP que está sendo usada é maior que a número de versão fornecido.

Exemplo:

if (is_php('5.3'))
{
        $str = quoted_printable_encode($str);
}

Retorna o booleano TRUE se a versão instalada do PHP for igual ou greater than the número de versão fornecido. Returns FALSE if the installed version of PHP is lower than the número de versão fornecido.

is_really_writable($file)¶
Parâmetros:
  • $file (string) - Caminho de arquivo
Retorna:

TRUE se o caminho for gravável, FALSE se não for
Tipo de retorno:

bool

is_writable() retorna TRUE em servidores Windows quando você realmente não pode escrever para o arquivo como o SO reporta ao PHP como FALSE somente se o o atributo somente leitura está marcado.

Esta função determina se um arquivo é realmente gravável tentando para escrever primeiro. Geralmente recomendado apenas em plataformas onde essa informação pode não ser confiável.

Exemplo:

if (is_really_writable('file.txt'))
{
        echo "I could write to this if I wanted to";
}
else
{
        echo "File is not writable";
}

Nota

Veja também Bug do PHP # 54709 para mais informações.

config_item($key)¶
Parâmetros:
  • $key (string) - Chave do item de configuração
Retorna:

Valor da chave de configuração ou NULL se não for encontrado
Tipo de retorno:

mixed

o Library de configuração é a maneira preferida de accessing configuration information, however config_item() can be used para recuperar chaves individuais. Vejo Library de configuração documentação para mais informações.

set_status_header($code[, $text = ''])¶
Parâmetros:
  • $code (int) - Código de status da resposta HTTP
  • $text (string) - Uma mensagem personalizada para definir com o código de status
Tipo de retorno:

void

Permite que você defina manualmente um cabeçalho de status do servidor. Exemplo:

set_status_header(401);
// Sets the header as:  Unauthorized

Veja aqui for uma lista completa de cabeçalhos.

remove_invisible_characters($str[, $url_encoded = TRUE])¶
Parâmetros:
  • $str (string) - string de entrada
  • $url_encoded (bool) - Se os caracteres codificados em URL também serão removidos
Retorna:

Corda higienizada
Tipo de retorno:

string

Esta função impede a inserção de caracteres NULL entre ASCII caracteres, como Java \ 0script.

Exemplo:

remove_invisible_characters('Java\\0script');
// Retorna: 'Javascript'
html_escape($var)¶
Parâmetros:
  • $var (mixed) - Variável para escapar (string ou array)
Retorna:

String (s) de escape em HTML
Tipo de retorno:

mixed

Esta função atua como um alias para o nativo do PHP htmlspecialchars() função, com a vantagem de poder aceitar um array de strings.

É útil na prevenção de Cross Site Scripting (XSS).

get_mimes()¶
Retorna:Uma matriz associativa de tipos de arquivos
Tipo de retorno:array

This function returns a reference to the MIMEs array from application/config/mimes.php.

is_https()¶
Retorna:Verdadeiro se atualmente usando HTTP-over-SSL, FALSE se não for
Tipo de retorno:bool

Retorna TRUE se uma conexão segura (HTTPS) for usada e FALSE em qualquer outro caso (incluindo solicitações não HTTP).

is_cli()¶
Retorna:Verdadeiro se atualmente em execução no CLI, FALSE caso contrário
Tipo de retorno:bool

Retorna TRUE se o aplicativo for executado por meio da linha de comando e FALSE se não.

Nota

This function checks both if the PHP_SAPI value is ‘cli’ or if the STDIN constant is defined.

function_usable($function_name)¶
Parâmetros:
  • $function_name (string) - nome da função
Retorna:

TRUE se a função puder ser usada, FALSE se não for
Tipo de retorno:

bool

Retorna TRUE se uma função existe e é utilizável, FALSE caso contrário.

This function runs a function_exists() check and if the Suhosin extension <http://www.hardened-php.net/suhosin/> está carregado, verifica se não desativa a função que está sendo verificada.

É útil se você quiser verificar a disponibilidade de funções such as eval() and exec(), which are dangerous and might be desativado em servidores com políticas de segurança altamente restritivas.

Nota

Esta função foi introduzida porque Suhosin terminou execução de script, mas isso acabou por ser um bug. Um conserto está disponível há algum tempo (versão 0.9.34), mas é infelizmente não foi lançado ainda.

Funções Comuns¶

O CodeIgniter usa algumas funções para sua operação que são globalmente definidos e estão disponíveis para você a qualquer momento. Estes não exigem carregando quaisquer librarys ou helpers.

is_php($version)¶
Parâmetros:
  • $version (string) - Número da versão
Retorna:

TRUE se a versão em execução do PHP é pelo menos aquela especificada ou FALSE se não for
Tipo de retorno:

bool

Determina se a versão do PHP que está sendo usada é maior que a número de versão fornecido.

Exemplo:

if (is_php('5.3'))
{
        $str = quoted_printable_encode($str);
}

Retorna o booleano TRUE se a versão instalada do PHP for igual ou greater than the número de versão fornecido. Returns FALSE if the installed version of PHP is lower than the número de versão fornecido.

is_really_writable($file)¶
Parâmetros:
  • $file (string) - Caminho de arquivo
Retorna:

TRUE se o caminho for gravável, FALSE se não for
Tipo de retorno:

bool

is_writable() retorna TRUE em servidores Windows quando você realmente não pode escrever para o arquivo como o SO reporta ao PHP como FALSE somente se o o atributo somente leitura está marcado.

Esta função determina se um arquivo é realmente gravável tentando para escrever primeiro. Geralmente recomendado apenas em plataformas onde essa informação pode não ser confiável.

Exemplo:

if (is_really_writable('file.txt'))
{
        echo "I could write to this if I wanted to";
}
else
{
        echo "File is not writable";
}

Nota

See also Bug do PHP # 54709 para mais informações.

config_item($key)¶
Parâmetros:
  • $key (string) - Chave do item de configuração
Retorna:

Valor da chave de configuração ou NULL se não for encontrado
Tipo de retorno:

mixed

o Library de configuração é a maneira preferida de accessing configuration information, however config_item() can be used para recuperar chaves individuais. Vejo Library de configuração documentação para mais informações.

set_status_header($code[, $text = ''])¶
Parâmetros:
  • $code (int) - Código de status da resposta HTTP
  • $text (string) - Uma mensagem personalizada para definir com o código de status
Tipo de retorno:

void

Permite que você defina manualmente um cabeçalho de status do servidor. Exemplo:

set_status_header(401);
// Sets the header as:  Unauthorized

See here for uma lista completa de cabeçalhos.

remove_invisible_characters($str[, $url_encoded = TRUE])¶
Parâmetros:
  • $str (string) - string de entrada
  • $url_encoded (bool) - Se os caracteres codificados em URL também serão removidos
Retorna:

Corda higienizada
Tipo de retorno:

string

Esta função impede a inserção de caracteres NULL entre ASCII caracteres, como Java \ 0script.

Exemplo:

remove_invisible_characters('Java\\0script');
// Retorna: 'Javascript'
html_escape($var)¶
Parâmetros:
  • $var (mixed) - Variável para escapar (string ou array)
Retorna:

String (s) de escape em HTML
Tipo de retorno:

mixed

Esta função atua como um alias para o nativo do PHP htmlspecialchars() função, com a vantagem de poder aceitar um array de strings.

É útil na prevenção de Cross Site Scripting (XSS).

get_mimes()¶
Retorna:Uma matriz associativa de tipos de arquivos
Tipo de retorno:array

This function returns a reference to the MIMEs array from application/config/mimes.php.

is_https()¶
Retorna:Verdadeiro se atualmente usando HTTP-over-SSL, FALSE se não for
Tipo de retorno:bool

Retorna TRUE se uma conexão segura (HTTPS) for usada e FALSE em qualquer outro caso (incluindo solicitações não HTTP).

is_cli()¶
Retorna:Verdadeiro se atualmente em execução no CLI, FALSE caso contrário
Tipo de retorno:bool

Retorna TRUE se o aplicativo for executado por meio da linha de comando e FALSE se não.

Nota

This function checks both if the PHP_SAPI value is ‘cli’ or if the STDIN constant is defined.

function_usable($function_name)¶
Parâmetros:
  • $function_name (string) - nome da função
Retorna:

TRUE se a função puder ser usada, FALSE se não for
Tipo de retorno:

bool

Retorna TRUE se uma função existe e é utilizável, FALSE caso contrário.

This function runs a function_exists() check and if the Suhosin extension <http://www.hardened-php.net/suhosin/> is loaded, verifica se não desativa a função que está sendo verificada.

É útil se você quiser verificar a disponibilidade de funções such as eval() and exec(), which are dangerous and might be desativado em servidores com políticas de segurança altamente restritivas.

Nota

Esta função foi introduzida porque Suhosin terminou execução de script, mas isso acabou por ser um bug. Um conserto está disponível há algum tempo (versão 0.9.34), mas é infelizmente não foi lançado ainda.
