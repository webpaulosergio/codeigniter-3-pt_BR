Funções de Compatibilidade¶

CodeIgniter provides a set of compatibility functions that enable você usar funções que estão nativamente disponíveis no PHP, but only in higher versions or depending on a certain extension.

Sendo implementações personalizadas, essas funções também terão algumas conjunto de dependências por conta própria, mas ainda são úteis se PHP setup doesn’t offer them natively.

Nota

Much like the Funções Comuns, the funções de compatibilidade estão sempre disponíveis, desde que their dependencies are met.

  • Hashing de Senha
    • Dependências
    • Constants
    • Referência de Função
  • Hash (Message Digest)
    • Dependências
    • Referência de Função
  • String Multibyte
    • Dependências
    • Referência de Função
  • Funções padrão
    • Dependências
    • Referência de Função

Hashing de Senha¶

This set of compatibility functions offers a “backport” of PHP’s standard Hashing de Senha extension that is otherwise available only since PHP 5.5.

Dependências¶

  • PHP 5.3.7
  • CRYPT_BLOWFISH suporte para crypt()

Constants¶

  • PASSWORD_BCRYPT
  • PASSWORD_DEFAULT

Referência de Função¶

password_get_info($hash)¶
Parâmetros:
  • $hash (string) – Password hash
Returns:

Information about the hashed password
Return type:

array

Para mais informações, por favor consulte o Manual do PHP para password_get_info().

password_hash($password, $algo[, $options = array()])¶
Parâmetros:
  • $password (string) – Plain-text password
  • $algo (int) - Algoritmo de Hashing
  • $options (array) - Opções de hash
Returns:

Hashed password or FALSE on failure
Return type:

string

Para mais informações, por favor consulte o Manual do PHP para password_hash().

Nota

A menos que você forneça seu próprio sal (e válido), essa função has a further dependency on an available CSPRNG source. Each of the following would satisfy that: - mcrypt_create_iv() with MCRYPT_DEV_URANDOM - openssl_random_pseudo_bytes() - /dev/arandom -/dev/urandom

password_needs_rehash()¶
Parâmetros:
  • $hash (string) – Password hash
  • $algo (int) - Algoritmo de Hashing
  • $options (array) - Opções de hash
Returns:

TRUE se o hash deve ser reescrito para coincidir com o algoritmo dado e opções, FALSE caso contrário
Return type:

bool

Para mais informações, por favor consulte o Manual do PHP para password_needs_rehash().

password_verify($password, $hash)¶
Parâmetros:
  • $password (string) – Plain-text password
  • $hash (string) – Password hash
Returns:

TRUE se a senha corresponder ao hash, FALSE se não for
Return type:

bool

Para mais informações, por favor consulte o Manual do PHP para password_verify().

Hash (Message Digest)¶

This compatibility layer contains backports for the hash_equals() and hash_pbkdf2() functions, which otherwise require PHP 5.6 and/or PHP 5.5 respectively.

Dependências¶

  • None

Referência de Função¶

hash_equals($known_string, $user_string)¶
Parâmetros:
  • $known_string (string) – Known string
  • $user_string (string) – User-supplied string
Returns:

TRUE if the strings match, FALSE otherwise
Return type:

string

Para mais informações, por favor consulte o Manual do PHP para hash_equals().

hash_pbkdf2($algo, $password, $salt, $iterations[, $length = 0[, $raw_output = FALSE]])¶
Parâmetros:
  • $algo (string) - Algoritmo de Hashing
  • $password (string) – Password
  • $salt (string) – Hash salt
  • $iterations (int) – Number of iterations to perform during derivation
  • $length (int) – Output string length
  • $raw_output (bool) - Se deve devolver dados binários brutos
Returns:

Chave derivada por senha ou FALSE em falha
Return type:

string

Para mais informações, por favor consulte o Manual do PHP para hash_pbkdf2().

String Multibyte¶

This set of compatibility functions offers limited suporte para PHP’s String Multibyte extension. Because of the limited alternative solutions, only a few functions are available.

Nota

Quando um parâmetro do conjunto de caracteres é omitido, $config['charset'] will be used.

Dependências¶

  • iconv extension

Importante

Essa dependência é opcional e essas funções always be declared. If iconv is not available, they WILL fall-back para suas versões não-mbstring.

Importante

Where a character set is supplied, it must be suportado por iconv e em um formato que reconhece.

Nota

For you own dependency check on the actual mbstring extension, use the MB_ENABLED constant.

Referência de Função¶

mb_strlen($str[, $encoding = NULL])¶
Parâmetros:
  • $str (string) – Input string
  • $encoding (string) – Character set
Returns:

Número de caracteres na string de entrada ou FALSE na falha
Return type:

string

Para mais informações, por favor consulte o Manual do PHP para mb_strlen().

mb_strpos($haystack, $needle[, $offset = 0[, $encoding = NULL]])¶
Parâmetros:
  • $haystack (string) – String to search in
  • $needle (string) – Part of string to search for
  • $offset (int) - Deslocamento de pesquisa
  • $encoding (string) – Character set
Returns:

Numeric character position of where $needle was found or FALSE if not found
Return type:

mixed

Para mais informações, por favor consulte o Manual do PHP para mb_strpos().

mb_substr($str, $start[, $length = NULL[, $encoding = NULL]])¶
Parâmetros:
  • $str (string) – Input string
  • $start (int) – Position of first character
  • $length (int) - Número máximo de caracteres
  • $encoding (string) – Character set
Returns:

Portion of $str specified by $start and $length or FALSE on failure
Return type:

string

Para mais informações, por favor consulte o Manual do PHP para mb_substr().

Funções padrão¶

This set of compatibility functions offers suporte para a few funções padrão em PHP que, de outra forma, requerem uma versão mais recente do PHP.

Dependências¶

  • None

Referência de Função¶

array_column(array $array, $column_key[, $index_key = NULL])¶
Parâmetros:
  • $array (array) - Matriz para buscar resultados de
  • $column_key (mixed) - Chave da coluna para retornar valores de
  • $index_key (mixed) – Key to use for the returned values
Returns:

An array of values representing a single column from the input array
Return type:

array

Para mais informações, por favor consulte o Manual do PHP para array_column().

hex2bin($data)¶
Parâmetros:
  • $data (array) - representação hexadecimal de dados
Returns:

Representação binária dos dados fornecidos
Return type:

string

Para mais informações, por favor consulte o Manual do PHP para hex2bin().

  • $user_string (string) - String fornecida pelo usuário
Retorna:

TRUE se as strings corresponderem, FALSE de outra forma
Tipo de retorno:

string

Para mais informações, por favor consulte o Manual do PHP para hash_equals().

hash_pbkdf2($algo, $password, $salt, $iterations[, $length = 0[, $raw_output = FALSE]])¶
Parâmetros:
  • $algo (string) - Algoritmo de Hashing
  • $password (string) - Senha

Funções de Compatibilidade¶

O CodeIgniter fornece um conjunto de funções de compatibilidade que permitem você usar funções que estão nativamente disponíveis no PHP, mas apenas em versões superiores ou dependendo de uma certa extensão.

Sendo implementações personalizadas, essas funções também terão algumas conjunto de dependências por conta própria, mas ainda são úteis se A configuração do PHP não oferece nativamente.

Nota

Muito parecido com o Funções Comuns, the funções de compatibilidade estão sempre disponíveis, desde que suas dependências são atendidas.

Hashing de Senha¶

Este conjunto de funções de compatibilidade oferece um "backport" de PHP standard Hashing de Senha extension que está disponível apenas desde o PHP 5.5.

Tipo de retorno:

string

A menos que você forneça seu próprio sal (e válido), essa função tem uma dependência adicional de uma fonte CSPRNG disponível. Cada dos seguintes seria satisfazer que: - mcrypt_create_iv() with MCRYPT_DEV_URANDOM - openssl_random_pseudo_bytes() -/dev/arandom -/dev/urandom
