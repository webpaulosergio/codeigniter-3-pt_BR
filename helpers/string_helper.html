Helper de Cordas¶

The Helper de Cordas file contains functions that assist in working with strings.

Importante

Por favor, note que estas funções não são intencionais, nem adequado para ser usado em qualquer tipo de lógica relacionada à segurança.

  • Loading this Helper
  • Funções Disponíveis

Loading this Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('string');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

random_string([$type = 'alnum'[, $len = 8]])¶
Parâmetros:
  • $type (string) – Randomization type
  • $len (int) – Output string length
Returns:

Uma string aleatória
Return type:

string

Generates a random string based on the type and length you specify. Useful for creating passwords or generating random hashes.

The first parameter specifies the type of string, the second parameter especifica o comprimento. As seguintes opções estão disponíveis:

  • alpha: Uma cadeia com apenas letras maiúsculas e minúsculas.
  • alnum: Seqüência alfanumérica com caracteres maiúsculos e minúsculos.
  • basic: A random number based on mt_rand().
  • numeric: Numeric string.
  • nozero: Cadeia numérica sem zeros.
  • md5: An encrypted random number based on md5() (fixed length of 32).
  • sha1: An encrypted random number based on sha1() (fixed length of 40).

Exemplo de uso:

echo random_string('alnum', 16);

Nota

Usage of the unique and encrypt types is DEPRECATED. They are just aliases for md5 and sha1 respectively.

increment_string($str[, $separator = '_'[, $first = 1]])¶
Parâmetros:
  • $str (string) – Input string
  • $separator (string) – Separator to append a duplicate number with
  • $first (int) – Starting number
Returns:

An incremented string
Return type:

string

Increments a string by appending a number to it or increasing the número. Útil para criar “cópias” ou um arquivo ou duplicar database conteúdo que possui títulos ou slugs únicos.

Exemplo de uso:

echo increment_string('file', '_'); // "file_1"
echo increment_string('file', '-', 2); // "file-2"
echo increment_string('file_4'); // "file_5"
alternator($args)¶
Parâmetros:
  • $args (mixed) – A variable number of arguments
Returns:

Corda (s) alternada (s)
Return type:

mixed

Permite que dois ou mais itens sejam alternados entre, quando passar de bicicleta uma volta. Exemplo:

for ($i = 0; $i < 10; $i++)
{
        echo alternator('string one', 'string two');
}

You can add as many parameters as you want, and with each iteration of seu loop o próximo item será retornado.

for ($i = 0; $i < 10; $i++)
{
        echo alternator('one', 'two', 'three', 'four', 'five');
}

Nota

To use multiple separate calls to this function simply call the função sem argumentos para reinicializar.

repeater($data[, $num = 1])¶
Parâmetros:
  • $data (string) – Input
  • $num (int) - Número de vezes para repetir
Returns:

Repeated string
Return type:

string

Generates repeating copies of the data you submit. Example:

$string = "\n";
echo repeater($string, 30);

O acima geraria 30 novas linhas.

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() instead.

reduce_double_slashes($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

Uma string com barras normalizadas
Return type:

string

Converte barras duplas em uma string em uma única barra, exceto aquelas encontrado em prefixos de protocolo de URL (por exemplo, http: //).

Example:

$string = "http://example.com//index.php";
echo reduce_double_slashes($string); // results in "http://example.com/index.php"
strip_slashes($data)¶
Parâmetros:
  • $data (mixed) – Input string or an array of strings
Returns:

String(s) with stripped slashes
Return type:

mixed

Removes any slashes from an array of strings.

Example:

$str = array(
        'question'  => 'Is your name O\'reilly?',
        'answer' => 'No, my name is O\'connor.'
);

$str = strip_slashes($str);

O acima irá retornar o seguinte array:

array(
        'question'  => "Is your name O'reilly?",
        'answer' => "No, my name is O'connor."
);

Nota

For historical reasons, this function will also accept and handle string inputs. This however makes it just an alias for stripslashes().

trim_slashes($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

Corda aparada
Return type:

string

Removes any leading/trailing slashes from a string. Example:

$string = "/this/that/theother/";
echo trim_slashes($string); // results in this/that/theother

Nota

Esta função é DEPRECADA. Use o nativo trim() instead: | | trim($str, ‘/’);

reduce_multiples($str[, $character = ''[, $trim = FALSE]])¶
Parâmetros:
  • $str (string) – Text to search in
  • $character (string) - Personagem para reduzir
  • $trim (bool) - Se também deve aparar o caractere especificado
Returns:

Corda Reduzida
Return type:

string

Reduces multiple instances of a particular character occurring directly after each other. Example:

$string = "Fred, Bill,, Joe, Jimmy";
$string = reduce_multiples($string,","); //results in "Fred, Bill, Joe, Jimmy"

If the third parameter is set to TRUE it will remove occurrences of the character at the beginning and the end of the string. Example:

$string = ",Fred, Bill,, Joe, Jimmy,";
$string = reduce_multiples($string, ", ", TRUE); //results in "Fred, Bill, Joe, Jimmy"
quotes_to_entities($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

String with quotes converted to HTML entities
Return type:

string

Converts single and double quotes in a string to the corresponding HTML entidades. Exemplo:

$string = "Joe's \"dinner\"";
$string = quotes_to_entities($string); //results in "Joe&#39;s &quot;dinner&quot;"
strip_quotes($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

String with quotes stripped
Return type:

string

Removes single and double quotes from a string. Example:

$string = "Joe's \"dinner\"";
$string = strip_quotes($string); //results in "Joes dinner"

Helper de Cordas¶

The Helper de Cordas file contains functions that assist in working with strings.

Important

Por favor, note que estas funções não são intencionais, nem adequado para ser usado em qualquer tipo de lógica relacionada à segurança.

  • Carregando este Helper
  • Funções Disponíveis

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('string');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

random_string([$type = 'alnum'[, $len = 8]])¶
Parâmetros:
  • $type (string) - tipo de randomização
  • $len (int) - Comprimento da string de saída
Retorna:

Uma string aleatória
Tipo de retorno:

string

Gera uma string aleatória com base no tipo e no comprimento que você especifica. Útil para criar senhas ou gerar hashes aleatórios.

O primeiro parâmetro especifica o tipo de string, o segundo parâmetro especifica o comprimento. As seguintes opções estão disponíveis:

  • alpha: Uma cadeia com apenas letras maiúsculas e minúsculas.
  • alnum: Seqüência alfanumérica com caracteres maiúsculos e minúsculos.
  • basic: A random number based on mt_rand().
  • numeric: String numérica.
  • nozero: Cadeia numérica sem zeros.
  • md5: An encrypted random number based on md5() (fixed length of 32).
  • sha1: An encrypted random number based on sha1() (fixed length of 40).

Exemplo de uso:

echo random_string('alnum', 16);

Nota

Usage of the unique and encrypt types is DEPRECATED. They are just aliases for md5 and sha1 respectively.

increment_string($str[, $separator = '_'[, $first = 1]])¶
Parâmetros:
  • $str (string) - string de entrada
  • $separator (string) - Separador para anexar um número duplicado com
  • $first (int) - número inicial
Retorna:

Uma string incrementada
Tipo de retorno:

string

Incrementa uma string anexando um número a ela ou aumentando número. Útil para criar "cópias" ou um arquivo ou duplicar database conteúdo que possui títulos ou slugs únicos.

Exemplo de uso:

echo increment_string('file', '_'); // "file_1"
echo increment_string('file', '-', 2); // "file-2"
echo increment_string('file_4'); // "file_5"
alternator($args)¶
Parâmetros:
  • $args (mixed) - Um número variável de argumentos
Retorna:

Corda (s) alternada (s)
Tipo de retorno:

mixed

Permite que dois ou mais itens sejam alternados entre, quando passar de bicicleta uma volta. Exemplo:

for ($i = 0; $i < 10; $i++)
{
        echo alternator('string one', 'string two');
}

Você pode adicionar quantos parâmetros desejar e a cada iteração de seu loop o próximo item será retornado.

for ($i = 0; $i < 10; $i++)
{
        echo alternator('one', 'two', 'three', 'four', 'five');
}

Nota

Para usar várias chamadas separadas para essa função, basta ligar para função sem argumentos para reinicializar.

repeater($data[, $num = 1])¶
Parâmetros:
  • $data (string) – Input
  • $num (int) - Número de vezes para repetir
Retorna:

Corda repetida
Tipo de retorno:

string

Gera cópias repetidas dos dados enviados. Exemplo:

$string = "\n";
echo repeater($string, 30);

O acima geraria 30 novas linhas.

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() instead.

reduce_double_slashes($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Uma string com barras normalizadas
Tipo de retorno:

string

Converte barras duplas em uma string em uma única barra, exceto aquelas encontrado em prefixos de protocolo de URL (por exemplo, http: //).

Exemplo:

$string = "http://example.com//index.php";
echo reduce_double_slashes($string); // results in "http://example.com/index.php"
