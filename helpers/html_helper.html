HTML Helper¶

The HTML Helper file contains functions that assist in working with HTML.

  • Loading this Helper
  • Funções Disponíveis

Loading this Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('html');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

heading([$data = ''[, $h = '1'[, $attributes = '']]])¶
Parâmetros:
  • $data (string) – Content
  • $h (string) - nível de rubrica
  • $attributes (mixed) - atributos HTML
Returns:

HTML heading tag
Return type:

string

Permite criar tags de cabeçalho HTML. O primeiro parâmetro conterá o dados, o segundo o tamanho do cabeçalho. Exemplo:

echo heading('Welcome!', 3);

The above would produce: <h3>Welcome!</h3>

Além disso, para adicionar atributos à tag de cabeçalho, como HTML classes, ids or inline styles, a third parameter accepts either a string or an array:

echo heading('Welcome!', 3, 'class="pink"');
echo heading('How are you?', 4, array('id' => 'question', 'class' => 'green'));

The above code produces:

<h3 class="pink">Welcome!<h3>
<h4 id="question" class="green">How are you?</h4>
img([$src = ''[, $index_page = FALSE[, $attributes = '']]])¶
Parâmetros:
  • $src (string) – Image source data
  • $index_page (bool) – Whether to treat $src as a routed URI string
  • $attributes (array) - atributos HTML
Returns:

HTML image tag
Return type:

string

Lets you create HTML <img /> tags. The first parameter contains the image source. Example:

echo img('images/picture.jpg'); // gives <img src="http://site.com/images/picture.jpg" />

Existe um segundo parâmetro opcional que é um valor TRUE/FALSE que specifics if the src should have the page specified by $config['index_page'] adicionado ao endereço que cria. Presumably, this would be if you were using a media controller:

echo img('images/picture.jpg', TRUE); // gives <img src="http://site.com/index.php/images/picture.jpg" alt="" />

Additionally, an associative array can be passed to the img() function for complete control over all attributes and values. If an alt attribute is not provided, CodeIgniter will generate an empty string.

Example:

$image_properties = array(
        'src'   => 'images/picture.jpg',
        'alt'   => 'Me, demonstrating how to eat 4 slices of pizza at one time',
        'class' => 'post_images',
        'width' => '200',
        'height'=> '200',
        'title' => 'That was quite a night',
        'rel'   => 'lightbox'
);

img($image_properties);
// <img src="http://site.com/index.php/images/picture.jpg" alt="Me, demonstrating how to eat 4 slices of pizza at one time" class="post_images" width="200" height="200" title="That was quite a night" rel="lightbox" />
link_tag([$href = ''[, $rel = 'stylesheet'[, $type = 'text/css'[, $title = ''[, $media = ''[, $index_page = FALSE]]]]]])¶
Parâmetros:
  • $href (string) – What are we linking to
  • $rel (string) – Relation type
  • $type (string) – Type of the related document
  • $title (string) – Link title
  • $media (string) - Tipo de mídia
  • $index_page (bool) – Whether to treat $src as a routed URI string
Returns:

HTML link tag
Return type:

string

Lets you create HTML <link /> tags. This is useful for stylesheet links, as well as other links. The parameters are href, with optional rel, type, title, media and index_page.

index_page is a boolean value that specifies if the href should have the page specified by $config['index_page'] adicionado ao endereço que cria.

Example:

echo link_tag('css/mystyles.css');
// gives <link href="http://site.com/css/mystyles.css" rel="stylesheet" type="text/css" />

Further examples:

echo link_tag('favicon.ico', 'shortcut icon', 'image/ico');
// <link href="http://site.com/favicon.ico" rel="shortcut icon" type="image/ico" />

echo link_tag('feed', 'alternate', 'application/rss+xml', 'My RSS Feed');
// <link href="http://site.com/feed" rel="alternate" type="application/rss+xml" title="My RSS Feed" />

Additionally, an associative array can be passed to the link() function for complete control over all attributes and values:

$link = array(
        'href'  => 'css/printer.css',
        'rel'   => 'stylesheet',
        'type'  => 'text/css',
        'media' => 'print'
);

echo link_tag($link);
// <link href="http://site.com/css/printer.css" rel="stylesheet" type="text/css" media="print" />
ul($list[, $attributes = ''])¶
Parâmetros:
  • $list (array) – List entries
  • $attributes (array) - atributos HTML
Returns:

HTML-formatted unordered list
Return type:

string

Permits you to generate unordered HTML lists from simple or multi-dimensional arrays. Example:

$list = array(
        'red',
        'blue',
        'green',
        'yellow'
);

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

echo ul($list, $attributes);

O código acima irá produzir isso:

<ul class="boldlist" id="mylist">
        <li>red</li>
        <li>blue</li>
        <li>green</li>
        <li>yellow</li>
</ul>

Aqui está um exemplo mais complexo, usando uma matriz multidimensional:

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

$list = array(
        'colors'  => array(
                'red',
                'blue',
                'green'
        ),
        'shapes'  => array(
                'round',
                'square',
                'circles' => array(
                        'ellipse',
                        'oval',
                        'sphere'
                )
        ),
        'moods'  => array(
                'happy',
                'upset' => array(
                        'defeated' => array(
                                'dejected',
                                'disheartened',
                                'depressed'
                        ),
                        'annoyed',
                        'cross',
                        'angry'
                )
        )
);

echo ul($list, $attributes);

O código acima irá produzir isso:

<ul class="boldlist" id="mylist">
        <li>colors
                <ul>
                        <li>red</li>
                        <li>blue</li>
                        <li>green</li>
                </ul>
        </li>
        <li>shapes
                <ul>
                        <li>round</li>
                        <li>suare</li>
                        <li>circles
                                <ul>
                                        <li>elipse</li>
                                        <li>oval</li>
                                        <li>sphere</li>
                                </ul>
                        </li>
                </ul>
        </li>
        <li>moods
                <ul>
                        <li>happy</li>
                        <li>upset
                                <ul>
                                        <li>defeated
                                                <ul>
                                                        <li>dejected</li>
                                                        <li>disheartened</li>
                                                        <li>depressed</li>
                                                </ul>
                                        </li>
                                        <li>annoyed</li>
                                        <li>cross</li>
                                        <li>angry</li>
                                </ul>
                        </li>
                </ul>
        </li>
</ul>
ol($list, $attributes = '')¶
Parâmetros:
  • $list (array) – List entries
  • $attributes (array) - atributos HTML
Returns:

HTML-formatted ordered list
Return type:

string

Identical to ul(), only it produces the <ol> tag for ordered lists instead of <ul>.

meta([$name = ''[, $content = ''[, $type = 'name'[, $newline = "n"]]]])¶
Parâmetros:
  • $name (string) – Meta name
  • $content (string) – Meta content
  • $type (string) – Meta type
  • $newline (string) – Newline character
Returns:

Meta tag HTML
Return type:

string

Ajuda você a gerar metatags. Você pode passar strings para a função ou arrays simples ou multidimensionais.

Exemplos:

echo meta('description', 'My Great site');
// Generates:  <meta name="description" content="My Great Site" />

echo meta('Content-type', 'text/html; charset=utf-8', 'equiv');
// Nota the third parameter.  Can be "equiv" or "name"
// Generates:  <meta http-equiv="Content-type" content="text/html; charset=utf-8" />

echo meta(array('name' => 'robots', 'content' => 'no-cache'));
// Generates:  <meta name="robots" content="no-cache" />

$meta = array(
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'description',
                'content' => 'My Great Site'
        ),
        array(
                'name' => 'keywords',
                'content' => 'love, passion, intrigue, deception'
        ),
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'Content-type',
                'content' => 'text/html; charset=utf-8', 'type' => 'equiv'
        )
);

echo meta($meta);
// Generates:
// <meta name="robots" content="no-cache" />
// <meta name="description" content="My Great Site" />
// <meta name="keywords" content="love, passion, intrigue, deception" />
// <meta name="robots" content="no-cache" />
// <meta http-equiv="Content-type" content="text/html; charset=utf-8" />
doctype([$type = 'xhtml1-strict'])¶
Parâmetros:
  • $type (string) – Doctype name
Returns:

HTML DocType tag
Return type:

string

Helps you generate document type declarations, or DTD’s. XHTML 1.0 Estrito é usado por padrão, mas muitos doctypes estão disponíveis.

Example:

echo doctype(); // <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

echo doctype('html4-trans'); // <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

br([$count = 1])¶
Parâmetros:
  • $count (int) - Número de vezes para repetir a tag
Returns:

Tag de quebra de linha HTML
Return type:

string

Generates line break tags (<br />) based on the number you submit. Example:

echo br(3);

The above would produce:

<br /><br /><br />

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() in combination with <br /> instead.

nbs([$num = 1])¶
Parâmetros:
  • $num (int) – Number of space entities to produce
Returns:

Uma sequência de entidades HTML de espaço sem quebra
Return type:

string

Generates non-breaking spaces (&nbsp;) based on the number you submit. Example:

echo nbs(3);

The above would produce:

&nbsp;&nbsp;&nbsp;

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() in combination with &nbsp; instead.

Helper HTML¶

The Helper HTML file contains functions that assist in working with HTML.

  • Carregando este Helper
  • Funções Disponíveis

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('html');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

heading([$data = ''[, $h = '1'[, $attributes = '']]])¶
Parâmetros:
  • $data (string) - Conteúdo
  • $h (string) - nível de rubrica
  • $attributes (mixed) - atributos HTML
Retorna:

Tag de cabeçalho HTML
Tipo de retorno:

string

Lets you create Tag de cabeçalho HTMLs. The first parameter will contain the dados, o segundo o tamanho do cabeçalho. Exemplo:

echo heading('Bem vinda!', 3);

O acima iria produzir: <h3>Bem vinda!</h3>

Além disso, para adicionar atributos à tag de cabeçalho, como HTML classes, ids ou inline styles, um terceiro parâmetro aceita uma string ou uma matriz:

echo heading('Bem vinda!', 3, 'class="pink"');
echo heading('Como você está?', 4, array('id' => 'question', 'class' => 'green'));

O código acima produz:

<h3 class="pink">Bem vinda!<h3>
<h4 id="question" class="green">Como você está?</h4>
img([$src = ''[, $index_page = FALSE[, $attributes = '']]])¶
Parâmetros:
  • $src (string) - Dados da fonte da imagem
  • $index_page (bool) - Para tratar $src como uma string URI roteada
  • $attributes (array) - atributos HTML
Retorna:

Tag de imagem HTML
Tipo de retorno:

string

Lets you create HTML <img /> tags. The first parameter contains the fonte de imagem. Exemplo:

echo img('images/picture.jpg'); // gives <img src="http://site.com/images/picture.jpg" />

Existe um segundo parâmetro opcional que é um valor TRUE/FALSE que specifics if the src should have the page specified by $config['index_page'] adicionado ao endereço que cria. Presumivelmente, isso seria se você estivesse usando um Controller de mídia:

echo img('images/picture.jpg', TRUE); // gives <img src="http://site.com/index.php/images/picture.jpg" alt="" />

Additionally, an associative array can be passed to the img() function for complete control over all attributes and values. If an alt attribute não é fornecido, o CodeIgniter irá gerar uma string vazia.

Exemplo:

$image_properties = array(
        'src'   => 'images/picture.jpg',
        'alt'   => 'Me, demonstrating how to eat 4 slices of pizza at one time',
        'class' => 'post_images',
        'width' => '200',
        'height'=> '200',
        'title' => 'That was quite a night',
        'rel'   => 'lightbox'
);

img($image_properties);
// <img src="http://site.com/index.php/images/picture.jpg" alt="Me, demonstrating how to eat 4 slices of pizza at one time" class="post_images" width="200" height="200" title="That was quite a night" rel="lightbox" />
link_tag([$href = ''[, $rel = 'stylesheet'[, $type = 'text/css'[, $title = ''[, $media = ''[, $index_page = FALSE]]]]]])¶
Parâmetros:
  • $href (string) - O que estamos ligando para
  • $rel (string) - tipo de relação
  • $type (string) - Tipo do documento relacionado
  • $title (string) - Título do link
  • $media (string) - Tipo de mídia
  • $index_page (bool) - Para tratar $src como uma string URI roteada
Retorna:

Tag de link HTML
Tipo de retorno:

string

Lets you create HTML <link /> tags. This is useful for stylesheet links, as well as other links. The parameters are href, with optional rel, type, title, media and index_page.

index_page is a boolean value that specifies if the href should have the page specified by $config['index_page'] adicionado ao endereço que cria.

Exemplo:

echo link_tag('css/mystyles.css');
// gives <link href="http://site.com/css/mystyles.css" rel="stylesheet" type="text/css" />

Mais exemplos:

echo link_tag('favicon.ico', 'shortcut icon', 'image/ico');
// <link href="http://site.com/favicon.ico" rel="shortcut icon" type="image/ico" />

echo link_tag('feed', 'alternate', 'application/rss+xml', 'My RSS Feed');
// <link href="http://site.com/feed" rel="alternate" type="application/rss+xml" title="My RSS Feed" />

Additionally, an associative array can be passed to the link() function para controle completo sobre todos os atributos e valores:

$link = array(
        'href'  => 'css/printer.css',
        'rel'   => 'stylesheet',
        'type'  => 'text/css',
        'media' => 'print'
);

echo link_tag($link);
// <link href="http://site.com/css/printer.css" rel="stylesheet" type="text/css" media="print" />
ul($list[, $attributes = ''])¶
Parâmetros:
  • $list (array) - Listar entradas
  • $attributes (array) - atributos HTML
Retorna:

Lista não ordenada formatada em HTML
Tipo de retorno:

string

Permite-lhe gerar listas HTML não ordenadas a partir de simples ou arrays multidimensionais. Exemplo:

$list = array(
        'red',
        'blue',
        'green',
        'yellow'
);

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

echo ul($list, $attributes);

O código acima irá produzir isso:

<ul class="boldlist" id="mylist">
        <li>red</li>
        <li>blue</li>
        <li>green</li>
        <li>yellow</li>
</ul>

Aqui está um exemplo mais complexo, usando uma matriz multidimensional:

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

$list = array(
        'colors'  => array(
                'red',
                'blue',
                'green'
        ),
        'shapes'  => array(
                'round',
                'square',
                'circles' => array(
                        'ellipse',
                        'oval',
                        'sphere'
                )
        ),
        'moods'  => array(
                'happy',
                'upset' => array(
                        'defeated' => array(
                                'dejected',
                                'disheartened',
                                'depressed'
                        ),
                        'annoyed',
                        'cross',
                        'angry'
                )
        )
);

echo ul($list, $attributes);

O código acima irá produzir isso:

<ul class="boldlist" id="mylist">
        <li>colors
                <ul>
                        <li>red</li>
                        <li>blue</li>
                        <li>green</li>
                </ul>
        </li>
        <li>shapes
                <ul>
                        <li>round</li>
                        <li>suare</li>
                        <li>circles
                                <ul>
                                        <li>elipse</li>
                                        <li>oval</li>
                                        <li>sphere</li>
                                </ul>
                        </li>
                </ul>
        </li>
        <li>moods
                <ul>
                        <li>happy</li>
                        <li>upset
                                <ul>
                                        <li>defeated
                                                <ul>
                                                        <li>dejected</li>
                                                        <li>disheartened</li>
                                                        <li>depressed</li>
                                                </ul>
                                        </li>
                                        <li>annoyed</li>
                                        <li>cross</li>
                                        <li>angry</li>
                                </ul>
                        </li>
                </ul>
        </li>
</ul>
ol($list, $attributes = '')¶
Parâmetros:
  • $list (array) - Listar entradas
  • $attributes (array) - atributos HTML
Retorna:

Lista ordenada formatada em HTML
Tipo de retorno:

string

Idêntico a ul(), only it produces the <ol> tag for ordered lists instead of <ul>.

meta([$name = ''[, $content = ''[, $type = 'name'[, $newline = "n"]]]])¶
Parâmetros:
  • $name (string) - Meta nome
  • $content (string) - conteúdo Meta
  • $type (string) - tipo Meta
  • $newline (string) - personagem Newline
Retorna:

Meta tag HTML
Tipo de retorno:

string

Ajuda você a gerar metatags. Você pode passar strings para a função ou arrays simples ou multidimensionais.

Exemplos:

echo meta('description', 'My Great site');
// Generates:  <meta name="description" content="My Great Site" />

echo meta('Content-type', 'text/html; charset=utf-8', 'equiv');
// Nota the third parameter.  Can be "equiv" or "name"
// Generates:  <meta http-equiv="Content-type" content="text/html; charset=utf-8" />

echo meta(array('name' => 'robots', 'content' => 'no-cache'));
// Generates:  <meta name="robots" content="no-cache" />

$meta = array(
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'description',
                'content' => 'My Great Site'
        ),
        array(
                'name' => 'keywords',
                'content' => 'love, passion, intrigue, deception'
        ),
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'Content-type',
                'content' => 'text/html; charset=utf-8', 'type' => 'equiv'
        )
);

echo meta($meta);
// Generates:
// <meta name="robots" content="no-cache" />
// <meta name="description" content="My Great Site" />
// <meta name="keywords" content="love, passion, intrigue, deception" />
// <meta name="robots" content="no-cache" />
// <meta http-equiv="Content-type" content="text/html; charset=utf-8" />
doctype([$type = 'xhtml1-strict'])¶
Parâmetros:
  • $type (string) - nome do tipo de documento
Retorna:

Tag DocType HTML
Tipo de retorno:

string

Ajuda você a gerar declarações de tipo de documento ou DTDs. XHTML 1.0 Estrito é usado por padrão, mas muitos doctypes estão disponíveis.

Exemplo:

echo doctype(); // <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

echo doctype('html4-trans'); // <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

br([$count = 1])¶
Parâmetros:
  • $count (int) - Número de vezes para repetir a tag
Retorna:

Tag de quebra de linha HTML
Tipo de retorno:

string

Generates line break tags (<br />) based on the number you submit. Exemplo:

echo br(3);

O acima iria produzir:

<br /><br /><br />

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() in combination with <br /> instead.

nbs([$num = 1])¶
Parâmetros:
  • $num (int) - Número de entidades espaciais para produzir
Retorna:

Uma sequência de entidades HTML de espaço sem quebra
Tipo de retorno:

string

Generates non-breaking spaces (&nbsp;) based on the number you submit. Exemplo:

echo nbs(3);

O acima iria produzir:

&nbsp;&nbsp;&nbsp;

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() in combination with &nbsp; instead.

Helper HTML¶

The Helper HTML file contains functions that assist in working with HTML.

  • Carregando este Helper
  • Funções Disponíveis

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('html');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

heading([$data = ''[, $h = '1'[, $attributes = '']]])¶
Parâmetros:
  • $data (string) - Conteúdo
  • $h (string) - nível de rubrica
  • $attributes (mixed) - atributos HTML
Retorna:

Tag de cabeçalho HTML
Tipo de retorno:

string

Lets you create Tag de cabeçalho HTMLs. The first parameter will contain the dados, o segundo o tamanho do cabeçalho. Exemplo:

echo heading('Welcome!', 3);

O acima iria produzir: <h3> Bem-vindo! </ H3>

Além disso, para adicionar atributos à tag de cabeçalho, como HTML classes, ids ou inline styles, um terceiro parâmetro aceita uma string ou uma matriz:

echo heading('Welcome!', 3, 'class="pink"');
echo heading('Como você está?', 4, array('id' => 'question', 'class' => 'green'));

O código acima produz:

<h3 class="pink">Welcome!<h3>
<h4 id="question" class="green">Como você está?</h4>
img([$src = ''[, $index_page = FALSE[, $attributes = '']]])¶
Parâmetros:
  • $src (string) - Dados da fonte da imagem
  • $index_page (bool) - Para tratar $src como uma string URI roteada
  • $attributes (array) - atributos HTML
Retorna:

Tag de imagem HTML
Tipo de retorno:

string

Permite criar tags HTML <img />. O primeiro parâmetro contém o fonte de imagem. Exemplo:

echo img('images/picture.jpg'); // gives <img src="http://site.com/images/picture.jpg" />

Existe um segundo parâmetro opcional que é um valor TRUE/FALSE que specifics if the src should have the page specified by $config['index_page'] adicionado ao endereço que cria. Presumivelmente, isso seria se você estivesse usando um Controller de mídia:

echo img('images/picture.jpg', TRUE); // gives <img src="http://site.com/index.php/images/picture.jpg" alt="" />

Additionally, an associative array can be passed to the img() function for complete control over all attributes and values. If an alt attribute não é fornecido, o CodeIgniter irá gerar uma string vazia.

Exemplo:

$image_properties = array(
        'src'   => 'images/picture.jpg',
        'alt'   => 'Me, demonstrating how to eat 4 slices of pizza at one time',
        'class' => 'post_images',
        'width' => '200',
        'height'=> '200',
        'title' => 'That was quite a night',
        'rel'   => 'lightbox'
);

img($image_properties);
// <img src="http://site.com/index.php/images/picture.jpg" alt="Me, demonstrating how to eat 4 slices of pizza at one time" class="post_images" width="200" height="200" title="That was quite a night" rel="lightbox" />
link_tag([$href = ''[, $rel = 'stylesheet'[, $type = 'text/css'[, $title = ''[, $media = ''[, $index_page = FALSE]]]]]])¶
Parâmetros:
  • $href (string) - O que estamos ligando para
  • $rel (string) - tipo de relação
  • $type (string) - Tipo do documento relacionado
  • $title (string) - Título do link
  • $media (string) - Tipo de mídia
  • $index_page (bool) - Para tratar $src como uma string URI roteada
Retorna:

Tag de link HTML
Tipo de retorno:

string

Permite criar tags HTML <link />. Isso é útil para links de folhas de estilo, as well as other links. The parameters are href, with optional rel, type, title, media and index_page.

index_page is a boolean value that specifies if the href should have the page specified by $config['index_page'] adicionado ao endereço que cria.

Exemplo:

echo link_tag('css/mystyles.css');
// gives <link href="http://site.com/css/mystyles.css" rel="stylesheet" type="text/css" />

Mais exemplos:

echo link_tag('favicon.ico', 'shortcut icon', 'image/ico');
// <link href="http://site.com/favicon.ico" rel="shortcut icon" type="image/ico" />

echo link_tag('feed', 'alternate', 'application/rss+xml', 'My RSS Feed');
// <link href="http://site.com/feed" rel="alternate" type="application/rss+xml" title="My RSS Feed" />

Additionally, an associative array can be passed to the link() function para controle completo sobre todos os atributos e valores:

$link = array(
        'href'  => 'css/printer.css',
        'rel'   => 'stylesheet',
        'type'  => 'text/css',
        'media' => 'print'
);

echo link_tag($link);
// <link href="http://site.com/css/printer.css" rel="stylesheet" type="text/css" media="print" />
ul($list[, $attributes = ''])¶
Parâmetros:
  • $list (array) - Listar entradas
  • $attributes (array) - atributos HTML
Retorna:

Lista não ordenada formatada em HTML
Tipo de retorno:

string

Permite-lhe gerar listas HTML não ordenadas a partir de simples ou arrays multidimensionais. Exemplo:

$list = array(
        'red',
        'blue',
        'green',
        'yellow'
);

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

echo ul($list, $attributes);

O código acima irá produzir isso:

<ul class="boldlist" id="mylist">
        <li>red</li>
        <li>blue</li>
        <li>green</li>
        <li>yellow</li>
</ul>

Aqui está um exemplo mais complexo, usando uma matriz multidimensional:

$attributes = array(
        'class' => 'boldlist',
        'id'    => 'mylist'
);

$list = array(
        'colors'  => array(
                'red',
                'blue',
                'green'
        ),
        'shapes'  => array(
                'round',
                'square',
                'circles' => array(
                        'ellipse',
                        'oval',
                        'sphere'
                )
        ),
        'moods'  => array(
                'happy',
                'upset' => array(
                        'defeated' => array(
                                'dejected',
                                'disheartened',
                                'depressed'
                        ),
                        'annoyed',
                        'cross',
                        'angry'
                )
        )
);

echo ul($list, $attributes);

O código acima irá produzir isso:

<ul class="boldlist" id="mylist">
        <li>colors
                <ul>
                        <li>red</li>
                        <li>blue</li>
                        <li>green</li>
                </ul>
        </li>
        <li>shapes
                <ul>
                        <li>round</li>
                        <li>suare</li>
                        <li>circles
                                <ul>
                                        <li>elipse</li>
                                        <li>oval</li>
                                        <li>sphere</li>
                                </ul>
                        </li>
                </ul>
        </li>
        <li>moods
                <ul>
                        <li>happy</li>
                        <li>upset
                                <ul>
                                        <li>defeated
                                                <ul>
                                                        <li>dejected</li>
                                                        <li>disheartened</li>
                                                        <li>depressed</li>
                                                </ul>
                                        </li>
                                        <li>annoyed</li>
                                        <li>cross</li>
                                        <li>angry</li>
                                </ul>
                        </li>
                </ul>
        </li>
</ul>

meta([$name = ''[, $content = ''[, $type = 'name'[, $newline = "n"]]]])¶
Parâmetros:
  • $name (string) - Meta nome
  • $content (string) - conteúdo Meta
  • $type (string) - tipo Meta
  • $newline (string) - personagem Newline
Retorna:

Meta tag HTML
Tipo de retorno:

string

Ajuda você a gerar metatags. Você pode passar strings para a função ou arrays simples ou multidimensionais.

Examples:

echo meta('description', 'My Great site');
// Generates:  <meta name="description" content="My Great Site" />

echo meta('Content-type', 'text/html; charset=utf-8', 'equiv');
// Nota the third parameter.  Can be "equiv" or "name"
// Generates:  <meta http-equiv="Content-type" content="text/html; charset=utf-8" />

echo meta(array('name' => 'robots', 'content' => 'no-cache'));
// Generates:  <meta name="robots" content="no-cache" />

$meta = array(
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'description',
                'content' => 'My Great Site'
        ),
        array(
                'name' => 'keywords',
                'content' => 'love, passion, intrigue, deception'
        ),
        array(
                'name' => 'robots',
                'content' => 'no-cache'
        ),
        array(
                'name' => 'Content-type',
                'content' => 'text/html; charset=utf-8', 'type' => 'equiv'
        )
);

echo meta($meta);
// Generates:
// <meta name="robots" content="no-cache" />
// <meta name="description" content="My Great Site" />
// <meta name="keywords" content="love, passion, intrigue, deception" />
// <meta name="robots" content="no-cache" />
// <meta http-equiv="Content-type" content="text/html; charset=utf-8" />
doctype([$type = 'xhtml1-strict'])¶
Parâmetros:
  • $type (string) - nome do tipo de documento
Retorna:

Tag DocType HTML
Tipo de retorno:

string

Ajuda você a gerar declarações de tipo de documento ou DTDs. XHTML 1.0 Estrito é usado por padrão, mas muitos doctypes estão disponíveis.

Exemplo:

echo doctype(); // <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

echo doctype('html4-trans'); // <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

br([$count = 1])¶
Parâmetros:
  • $count (int) - Número de vezes para repetir a tag
Retorna:

Tag de quebra de linha HTML
Tipo de retorno:

string

Gera tags de quebra de linha (<br />) com base no número que você envia. Exemplo:

echo br(3);

O acima iria produzir:

<br /><br /><br />

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() in combination with <br /> instead.

nbs([$num = 1])¶
Parâmetros:
  • $num (int) - Número de entidades espaciais para produzir
Retorna:

Uma sequência de entidades HTML de espaço sem quebra
Tipo de retorno:

string

Gera espaços não separáveis (& nbsp;) com base no número que você envia. Exemplo:

echo nbs(3);

O acima iria produzir:

&nbsp;&nbsp;&nbsp;

Nota

Esta função é DEPRECADA. Use o nativo str_repeat() in combination with &nbsp; instead.
