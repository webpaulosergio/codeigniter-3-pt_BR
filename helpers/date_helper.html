Helper de Data¶

The Helper de Data file contains functions that help you work with dates.

  • Loading this Helper
  • Funções Disponíveis
  • Timezone Reference

Loading this Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('date');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

now([$timezone = NULL])¶
Parâmetros:
  • $timezone (string) - Fuso horário
Returns:

UNIX timestamp
Return type:

int

Retorna a hora atual como um timestamp do UNIX, referenciado para o seu servidor local time or any PHP supported timezone, based on the “time reference” setting no seu arquivo de configuração. Se você não pretende definir sua referência de horário mestre para any other PHP supported timezone (which you’ll typically do if you run a site that lets each user set their own timezone settings) there is no benefit to using this function over PHP’s time() function.

echo now('Australia/Victoria');

If a timezone is not provided, it will return time() based on the time_reference setting.

mdate([$datestr = ''[, $time = '']])¶
Parâmetros:
  • $datestr (string) – Date string
  • $time (int) - timestamp do UNIX
Returns:

MySQL-formatted date
Return type:

string

This function is identical to PHP’s date() function, except that it lets you use MySQL style date codes, where each code letter is preceded with a percent sign, e.g. %Y %m %d

The benefit of doing dates this way is that you don’t have to worry about escaping any characters that are not date codes, as you would normally have to do with the date() function.

Example:

$datestring = 'Year: %Y Month: %m Day: %d - %h:%i %a';
$time = time();
echo mdate($datestring, $time);

Se um timestamp não estiver incluído no segundo parâmetro, a hora atual will be used.

standard_date([$fmt = 'DATE_RFC822'[, $time = NULL]])¶
Parâmetros:
  • $fmt (string) – Date format
  • $time (int) - timestamp do UNIX
Returns:

Formatted date or FALSE on invalid format
Return type:

string

Permite gerar uma string de data em um dos vários formatos padronizados.

Example:

$format = 'DATE_RFC822';
$time = time();
echo standard_date($format, $time);

Nota

This function is DEPRECATED. Use the native date() combined with DateTime’s format constants instead:

echo date(DATE_RFC822, time());

Supported formats:

Constant Descrição Example
DATE_ATOM Atom 2005-08-15T16:13:03+0000
DATE_COOKIE HTTP Cookies Sun, 14 Aug 2005 16:13:03 UTC
DATE_ISO8601 ISO-8601 2005-08-14T16:13:03+00:00
DATE_RFC822 RFC 822 Dom, 14 de agosto 05 16:13:03 UTC
DATE_RFC850 RFC 850 Domingo, 14 de agosto de 2005 - 16:13:03 UTC
DATE_RFC1036 RFC 1036 Domingo, 14 de agosto de 2005 - 16:13:03 UTC
DATE_RFC1123 RFC 1123 Sun, 14 Aug 2005 16:13:03 UTC
DATE_RFC2822 RFC 2822 Sun, 14 Aug 2005 16:13:03 +0000
DATE_RSS RSS Sun, 14 Aug 2005 16:13:03 UTC
DATE_W3C W3C 2005-08-14T16:13:03+0000
local_to_gmt([$time = ''])¶
Parâmetros:
  • $time (int) - timestamp do UNIX
Returns:

UNIX timestamp
Return type:

int

Recebe um timestamp do UNIX como entrada e o retorna como GMT.

Example:

$gmt = local_to_gmt(time());
gmt_to_local([$time = ''[, $timezone = 'UTC'[, $dst = FALSE]]])¶
Parâmetros:
  • $time (int) - timestamp do UNIX
  • $timezone (string) - Fuso horário
  • $dst (bool) – Whether DST is active
Returns:

UNIX timestamp
Return type:

int

Takes a UNIX timestamp (referenced to GMT) as input, and converts it to um timestamp localizado com base no fuso horário e no horário de verão submitted.

Example:

$timestamp = 1140153693;
$timezone  = 'UM8';
$daylight_saving = TRUE;
echo gmt_to_local($timestamp, $timezone, $daylight_saving);

Nota

Para uma lista de fusos horários, consulte a referência na parte inferior desta página.

mysql_to_unix([$time = ''])¶
Parâmetros:
  • $time (string) – MySQL timestamp
Returns:

UNIX timestamp
Return type:

int

Takes a MySQL Timestamp as input and returns it as a UNIX timestamp.

Example:

$unix = mysql_to_unix('20061124092345');
unix_to_human([$time = ''[, $seconds = FALSE[, $fmt = 'us']]])¶
Parâmetros:
  • $time (int) - timestamp do UNIX
  • $seconds (bool) - Se mostrar segundos
  • $fmt (string) - formato (nós ou euro)
Returns:

Formatted date
Return type:

string

Takes a UNIX timestamp as input and returns it in a human readable format with this prototype:

YYYY-MM-DD HH:MM:SS AM/PM

This can be useful if you need to display a date in a form field for submission.

The time can be formatted with or without seconds, and it can be set to Formato europeu ou americano. Se somente o timestamp for enviado, ele retornará o tempo sem segundos formatado para os EUA

Exemplos:

$now = time();
echo unix_to_human($now); // U.S. time, no seconds
echo unix_to_human($now, TRUE, 'us'); // U.S. time with seconds
echo unix_to_human($now, TRUE, 'eu'); // Euro time with seconds
human_to_unix([$datestr = ''])¶
Parâmetros:
  • $datestr (int) – Date string
Returns:

Carimbo de data/hora do UNIX ou FALSE na falha
Return type:

int

The opposite of the unix_to_time() function. Takes a “human” time as input and returns it as a UNIX timestamp. This is useful if you accept “human” formatted dates submitted via a form. Returns boolean FALSE data string passada para ele não está formatada como indicado acima.

Example:

$now = time();
$human = unix_to_human($now);
$unix = human_to_unix($human);
nice_date([$bad_date = ''[, $format = FALSE]])¶
Parâmetros:
  • $bad_date (int) – The terribly formatted date-like string
  • $format (string) – Date format to return (same as PHP’s date() function)
Returns:

Formatted date
Return type:

string

Esta função pode ter um número de formatos de data malformados e converter them into something useful. It also accepts well-formed dates.

A função retornará um timestamp do UNIX por padrão. Você pode, opcionalmente, pass a format string (the same type as the PHP date() function accepts) como o segundo parâmetro.

Example:

$bad_date = '199605';
// Should Produce: 1996-05-01
$better_date = nice_date($bad_date, 'Y-m-d');

$bad_date = '9-11-2001';
// Should Produce: 2001-09-11
$better_date = nice_date($bad_date, 'Y-m-d');

Nota

This function is DEPRECATED. Use PHP’s native Classe DateTime instead.

timespan([$seconds = 1[, $time = ''[, $units = '']]])¶
Parâmetros:
  • $seconds (int) - número de segundos
  • $time (string) - timestamp do UNIX
  • $units (int) - Número de unidades de tempo a exibir
Returns:

Diferença de tempo formatado
Return type:

string

Formats a UNIX timestamp so that is appears similar to this:

1 Year, 10 Months, 2 Weeks, 5 Days, 10 Hours, 16 Minutes

The first parameter must contain a UNIX timestamp. The second parameter must contain a timestamp that is greater that the primeiro timestamp. The thirdparameter is optional and limits the number of time units to display.

If the second parameter empty, the current time will be used.

The most common purpose for this function is to show how much time has decorrido de algum ponto no tempo no passado até agora.

Example:

$post_date = '1079621429';
$now = time();
$units = 2;
echo timespan($post_date, $now, $units);

Nota

The text generated by this function is found in the following language file: language/<your_lang>/date_lang.php

days_in_month([$month = 0[, $year = '']])¶
Parâmetros:
  • $month (int) – a numeric month
  • $year (int) – a numeric year
Returns:

Count of days in the specified month
Return type:

int

Returns the number of days in a given month/year. Takes leap years into account.

Example:

echo days_in_month(06, 2005);

Se o segundo parâmetro estiver vazio, o ano atual será usado.

Nota

This function will alias the native cal_days_in_month(), if está disponível.

date_range([$unix_start = ''[, $mixed = ''[, $is_unix = TRUE[, $format = 'Y-m-d']]]])¶
Parâmetros:
  • $unix_start (int) - timestamp do UNIX of the range start date
  • $mixed (int) - timestamp do UNIX of the range end date or interval in days
  • $is_unix (bool) – set to FALSE if $mixed is not a timestamp
  • $format (string) - Formato de data de saída, igual ao de date()
Returns:

Uma matriz de datas
Return type:

array

Retorna uma lista de datas dentro de um período especificado.

Example:

$range = date_range('2012-01-01', '2012-01-15');
echo "First 15 days of 2012:";
foreach ($range as $date)
{
        echo $date."\n";
}
timezones([$tz = ''])¶
Parâmetros:
  • $tz (string) – A numeric timezone
Returns:

Hour difference from UTC
Return type:

int

Takes a timezone reference (for a list of valid timezones, see the “Timezone Reference” below) and returns the number of hours offset from UTC.

Example:

echo timezones('UM5');

Esta função é útil quando usada com timezone_menu().

timezone_menu([$default = 'UTC'[, $class = ''[, $name = 'timezones'[, $attributes = '']]]])¶
Parâmetros:
  • $default (string) - Fuso horário
  • $class (string) – Class name
  • $name (string) – Menu name
  • $attributes (mixed) - atributos HTML
Returns:

HTML drop down menu with time zones
Return type:

string

Generates a pull-down menu of timezones, like this one:

Este menu é útil se você executar um site de associação no qual seus usuários permissão para definir seu valor de fuso horário local.

The first parameter lets you set the “selected” state of the menu. For example, to set Pacific time as the default you will do this:

echo timezone_menu('UM8');

Please see the timezone reference below to see the values of this menu.

The second parameter lets you set a CSS class name for the menu.

O quarto parâmetro permite definir um ou mais atributos na tag de seleção gerada.

Nota

The text contained in the menu is found in the following language file: language/<your_lang>/date_lang.php

Helper de Data¶

The Helper de Data file contains functions that help you work with dates.

  • Carregando este Helper
  • Funções Disponíveis
  • Referência de fuso horário

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('date');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

now([$timezone = NULL])¶
Parâmetros:
  • $timezone (string) - Fuso horário
Retorna:

Timestamp do UNIX
Tipo de retorno:

int

Retorna a hora atual como um timestamp do UNIX, referenciado para o seu servidor hora local ou qualquer fuso horário suportado pelo PHP, com base na configuração de "referência de tempo" no seu arquivo de configuração. Se você não pretende definir sua referência de horário mestre para qualquer outro fuso horário suportado pelo PHP (o que você normalmente fará se você executar um site que permite que cada usuário defina suas próprias configurações de fuso horário) não há nenhum benefício em usar this function over PHP’s time() function.

echo now('Australia/Victoria');

If a timezone is not provided, it will return time() based on the time_reference setting.

mdate([$datestr = ''[, $time = '']])¶
Parâmetros:
  • $datestr (string) - String de data
  • $time (int) – Timestamp do UNIX
Retorna:

Data formatada no MySQL
Tipo de retorno:

string

Esta função é idêntica à do PHP date() função, exceto que ele permite usar códigos de data no estilo MySQL, onde cada a letra do código é precedida por um sinal de porcentagem, por exemplo % Y% m% d

O benefício de fazer datas dessa maneira é que você não precisa se preocupar sobre como escapar de quaisquer caracteres que não sejam códigos de data, como você faria normally have to do with the date() function.

Exemplo:

$datestring = 'Year: %Y Month: %m Day: %d - %h:%i %a';
$time = time();
echo mdate($datestring, $time);

Se um timestamp não estiver incluído no segundo parâmetro, a hora atual será usado.

standard_date([$fmt = 'DATE_RFC822'[, $time = NULL]])¶
Parâmetros:
  • $fmt (string) - Formato de data
  • $time (int) – Timestamp do UNIX
Retorna:

Data formatada ou FALSE em formato inválido
Tipo de retorno:

string

Permite gerar uma string de data em um dos vários formatos padronizados.

Exemplo:

$format = 'DATE_RFC822';
$time = time();
echo standard_date($format, $time);

Nota

This function is DEPRECATED. Use the native date() combined with Constantees de formato do DateTime instead:

echo date(DATE_RFC822, time());

Formatos Suportados:

Constante Descrição Exemplo
DATE_ATOM Atom 2005-08-15T16:13:03+0000
DATE_COOKIE Cookies HTTP Dom, 14 de agosto de 2005 16:13:03 UTC
DATE_ISO8601 ISO-8601 2005-08-14T16:13:03+00:00
DATE_RFC822 RFC 822 Dom, 14 de agosto 05 16:13:03 UTC
DATE_RFC850 RFC 850 Domingo, 14 de agosto de 2005 - 16:13:03 UTC
DATE_RFC1036 RFC 1036 Domingo, 14 de agosto de 2005 - 16:13:03 UTC
DATE_RFC1123 RFC 1123 Dom, 14 de agosto de 2005 16:13:03 UTC
DATE_RFC2822 RFC 2822 Sun, 14 Aug 2005 16:13:03 +0000
DATE_RSS RSS Dom, 14 de agosto de 2005 16:13:03 UTC
DATE_W3C W3C 2005-08-14T16:13:03+0000
local_to_gmt([$time = ''])¶
Parâmetros:
  • $time (int) – Timestamp do UNIX
Retorna:

Timestamp do UNIX
Tipo de retorno:

int

Recebe um timestamp do UNIX como entrada e o retorna como GMT.

Exemplo:

$gmt = local_to_gmt(time());
gmt_to_local([$time = ''[, $timezone = 'UTC'[, $dst = FALSE]]])¶
Parâmetros:
  • $time (int) – Timestamp do UNIX
  • $timezone (string) - Fuso horário
  • $dst (bool) - Se o DST está ativo
Retorna:

Timestamp do UNIX
Tipo de retorno:

int

Obtém um registro de data e hora do UNIX (referenciado ao GMT) como entrada e o converte em um timestamp localizado com base no fuso horário e no horário de verão submitted.

Exemplo:

$timestamp = 1140153693;
$timezone  = 'UM8';
$daylight_saving = TRUE;
echo gmt_to_local($timestamp, $timezone, $daylight_saving);

Nota

Para uma lista de fusos horários, consulte a referência na parte inferior desta página.

mysql_to_unix([$time = ''])¶
Parâmetros:
  • $time (string) - timestamp do MySQL
Retorna:

Timestamp do UNIX
Tipo de retorno:

int

Takes a MySQL Timestamp as input and returns it as a Timestamp do UNIX.

Exemplo:

$unix = mysql_to_unix('20061124092345');
unix_to_human([$time = ''[, $seconds = FALSE[, $fmt = 'us']]])¶
Parâmetros:
  • $time (int) – Timestamp do UNIX
  • $seconds (bool) - Se mostrar segundos
  • $fmt (string) - formato (nós ou euro)
Retorna:

Data formatada
Tipo de retorno:

string

Takes a Timestamp do UNIX as input and returns it in a human readable formato com este protótipo:

YYYY-MM-DD HH:MM:SS AM/PM

Isso pode ser útil se você precisar exibir uma data em um campo de formulário para submission.

A hora pode ser formatada com ou sem segundos e pode ser definida como Formato europeu ou americano. Se somente o timestamp for enviado, ele retornará o tempo sem segundos formatado para os EUA

Exemplos:

$now = time();
echo unix_to_human($now); // U.S. time, no seconds
echo unix_to_human($now, TRUE, 'us'); // U.S. time with seconds
echo unix_to_human($now, TRUE, 'eu'); // Euro time with seconds
human_to_unix([$datestr = ''])¶
Parâmetros:
  • $datestr (int) - String de data
Retorna:

Timestamp do UNIX or FALSE on failure
Tipo de retorno:

int

The opposite of the unix_to_time() function. Takes a “human” time as input and returns it as a Timestamp do UNIX. This is useful if you aceitar datas formatadas “humanas” enviadas por meio de um formulário. Retorna FALSE booleano data string passada para ele não está formatada como indicado acima.

Exemplo:

$now = time();
$human = unix_to_human($now);
$unix = human_to_unix($human);
nice_date([$bad_date = ''[, $format = FALSE]])¶
Parâmetros:
  • $bad_date (int) - A string de data-data terrivelmente formatada
  • $format (string) - Formato de data to return (same as PHP’s date() function)
Retorna:

Data formatada
Tipo de retorno:

string

Esta função pode ter um número de formatos de data malformados e converter -los em algo útil. Também aceita datas bem formadas.

The function will return a Timestamp do UNIX by default. You can, optionally, pass a format string (the same type as the PHP date() function accepts) como o segundo parâmetro.

Exemplo:

$bad_date = '199605';
// Should Produce: 1996-05-01
$better_date = nice_date($bad_date, 'Y-m-d');

$bad_date = '9-11-2001';
// Should Produce: 2001-09-11
$better_date = nice_date($bad_date, 'Y-m-d');

Nota

Esta função é DEPRECADA. Use o nativo do PHP Classe DateTime instead.

timespan([$seconds = 1[, $time = ''[, $units = '']]])¶
Parâmetros:
  • $seconds (int) - número de segundos
  • $time (string) – Timestamp do UNIX
  • $units (int) - Número de unidades de tempo a exibir
Retorna:

Diferença de tempo formatado
Tipo de retorno:

string

Formats a Timestamp do UNIX so that is appears similar to this:

1 Year, 10 Months, 2 Weeks, 5 Days, 10 Hours, 16 Minutes

The first parameter must contain a Timestamp do UNIX. O segundo parâmetro deve conter um registro de data e hora maior do que o primeiro timestamp. O terceiro parâmetro é opcional e limita o número de unidades de tempo a serem exibidas.

If the second parameter empty, the current time será usado.

O propósito mais comum para esta função é mostrar quanto tempo decorrido de algum ponto no tempo no passado até agora.

Exemplo:

$post_date = '1079621429';
$now = time();
$units = 2;
echo timespan($post_date, $now, $units);

Nota

O texto gerado por esta função é encontrado no seguinte idioma file: language/<your_lang>/date_lang.php

days_in_month([$month = 0[, $year = '']])¶
Parâmetros:
  • $month (int) - um mês numérico
  • $year (int) - um ano numérico
Retorna:

Contagem de dias no mês especificado
Tipo de retorno:

int

Retorna o número de dias em um determinado mês/ano. Leva anos bissextos account.

Exemplo:

echo days_in_month(06, 2005);

If the second parameter is empty, the current year será usado.

Nota

This function will alias the native cal_days_in_month(), if está disponível.

date_range([$unix_start = ''[, $mixed = ''[, $is_unix = TRUE[, $format = 'Y-m-d']]]])¶
Parâmetros:
  • $unix_start (int) – Timestamp do UNIX of the range start date
  • $mixed (int) – Timestamp do UNIX of the range end date or interval in days
  • $is_unix (bool) - defina como FALSE se $mixed não for um timestamp
  • $format (string) - Formato de data de saída, igual ao de date()
Retorna:

Uma matriz de datas
Tipo de retorno:

array

Retorna uma lista de datas dentro de um período especificado.

Exemplo:

$range = date_range('2012-01-01', '2012-01-15');
echo "First 15 days of 2012:";
foreach ($range as $date)
{
        echo $date."\n";
}
timezones([$tz = ''])¶
Parâmetros:
  • $tz (string) - um fuso horário numérico
Retorna:

Diferença de hora entre UTC
Tipo de retorno:

int

Toma uma referência de fuso horário (para uma lista de fusos horários válidos, consulte o “Referência de fuso horário” abaixo) e retorna o número de horas de deslocamento UTC.

Exemplo:

echo timezones('UM5');

Esta função é útil quando usada com timezone_menu().

timezone_menu([$default = 'UTC'[, $class = ''[, $name = 'timezones'[, $attributes = '']]]])¶
Parâmetros:
  • $default (string) - Fuso horário
  • $class (string) - Nome da classe
  • $name (string) - nome do menu
  • $attributes (mixed) - atributos HTML
Retorna:

Menu suspenso HTML com fusos horários
Tipo de retorno:

string

Gera um menu suspenso de fusos horários, como este:

Este menu é útil se você executar um site de associação no qual seus usuários permissão para definir seu valor de fuso horário local.

O primeiro parâmetro permite definir o estado “selecionado” do menu. Para Por exemplo, para definir a hora do Pacífico como padrão, você fará isso:

echo timezone_menu('UM8');

Por favor, veja a referência de fuso horário abaixo para ver os valores deste menu.

O segundo parâmetro permite definir um nome de classe CSS para o menu.

O quarto parâmetro permite definir um ou mais atributos na tag de seleção gerada.

Nota

O texto contido no menu é encontrado nas seguintes arquivo de idioma: language/<your_lang>/date_lang.php

Tipo de retorno:

string

Esta função é idêntica à do PHP date() função, exceto que ele permite usar códigos de data no estilo MySQL, onde cada a letra do código é precedida por um sinal de porcentagem, por exemplo %Y %m %d

O benefício de fazer datas dessa maneira é que você não precisa se preocupar sobre como escapar de quaisquer caracteres que não sejam códigos de data, como você faria normally have to do with the date() function.

Exemplo:

$datestring = 'Year: %Y Month: %m Day: %d - %h:%i %a';
$time = time();
echo mdate($datestring, $time);

Se um timestamp não estiver incluído no segundo parâmetro, a hora atual será usado.

unix_to_human([$time = ''[, $seconds = FALSE[, $fmt = 'us']]])¶
Parâmetros:
  • $time (int) – Timestamp do UNIX
  • $seconds (bool) - Se mostrar segundos
  • $fmt (string) - formato (nós ou euro)
Retorna:

Data formatada
Tipo de retorno:

string

Takes a Timestamp do UNIX as input and returns it in a human readable formato com este protótipo:

YYYY-MM-DD HH:MM:SS AM/PM

Isso pode ser útil se você precisar exibir uma data em um campo de formulário para submission.

A hora pode ser formatada com ou sem segundos e pode ser definida como Formato europeu ou americano. Se somente o timestamp for enviado, ele retornará o tempo sem segundos formatado para os EUA

Exemplos:

$now = time();
echo unix_to_human($now); // U.S. time, no seconds
echo unix_to_human($now, TRUE, 'us'); // U.S. time with seconds
echo unix_to_human($now, TRUE, 'eu'); // Euro time with seconds

nice_date([$bad_date = ''[, $format = FALSE]])¶
Parâmetros:
  • $bad_date (int) - A string de data-data terrivelmente formatada
  • $format (string) - Formato de data to return (same as PHP’s date() function)
Retorna:

Data formatada
Tipo de retorno:

string

Esta função pode ter um número de formatos de data malformados e converter -los em algo útil. Também aceita datas bem formadas.

The function will return a Timestamp do UNIX by default. You can, optionally, pass a format string (the same type as the PHP date() function accepts) como o segundo parâmetro.

Exemplo:

$bad_date = '199605';
// Should Produce: 1996-05-01
$better_date = nice_date($bad_date, 'Y-m-d');

$bad_date = '9-11-2001';
// Should Produce: 2001-09-11
$better_date = nice_date($bad_date, 'Y-m-d');

Nota

Esta função é DEPRECADA. Use o nativo do PHP Classe DateTime instead.

timespan([$seconds = 1[, $time = ''[, $units = '']]])¶
Parâmetros:
  • $seconds (int) - número de segundos
  • $time (string) – Timestamp do UNIX
  • $units (int) - Número de unidades de tempo a exibir
Retorna:

Diferença de tempo formatado
Tipo de retorno:

string

Formats a Timestamp do UNIX so that is appears similar to this:

1 Year, 10 Months, 2 Weeks, 5 Days, 10 Hours, 16 Minutes

The first parameter must contain a Timestamp do UNIX. O segundo parâmetro deve conter um registro de data e hora maior do que o primeiro timestamp. O terceiro parâmetro é opcional e limita o número de unidades de tempo a serem exibidas.

If the second parameter empty, the current time será usado.

O propósito mais comum para esta função é mostrar quanto tempo decorrido de algum ponto no tempo no passado até agora.

Exemplo:

$post_date = '1079621429';
$now = time();
$units = 2;
echo timespan($post_date, $now, $units);

Nota

O texto gerado por esta função é encontrado no seguinte idioma file: language/<your_lang>/date_lang.php
