CAPTCHA Helper¶

o CAPTCHA Helper file contains functions that assist in creating CAPTCHA images.

  • Loading this Helper
  • Using the CAPTCHA helper
    • Adding a Database
  • Funções Disponíveis

Loading this Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('captcha');

Using the CAPTCHA helper¶

Uma vez carregado, você pode gerar um CAPTCHA assim:

$vals = array(
        'word'          => 'Random word',
        'img_path'      => './captcha/',
        'img_url'       => 'http://example.com/captcha/',
        'font_path'     => './path/to/fonts/texb.ttf',
        'img_width'     => '150',
        'img_height'    => 30,
        'expiration'    => 7200,
        'word_length'   => 8,
        'font_size'     => 16,
        'img_id'        => 'Imageid',
        'pool'          => '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ',

        // White background and border, black text and red grid
        'colors'        => array(
                'background' => array(255, 255, 255),
                'border' => array(255, 255, 255),
                'text' => array(0, 0, 0),
                'grid' => array(255, 40, 40)
        )
);

$cap = create_captcha($vals);
echo $cap['image'];
  • o captcha function requires the GD image library.
  • Apenas o img_path and img_url é requerido.
  • Se um word is not supplied, the function will generate a random String ASCII. Você pode montar sua própria library de palavras que você can draw randomly from.
  • If you do not specify a path to a TRUE TYPE font, the native ugly GD font will be used.
  • o “captcha” directory must be writable
  • o expiration (in seconds) signifies how long an image will remain na pasta captcha antes de ser apagada. O padrão é dois hours.
  • word_length o padrão é 8 pool defaults to ‘0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ’
  • font_size defaults to 16, the native GD font has a size limit. Specify a “true type” font for bigger sizes.
  • o img_id será definido como o "id" da imagem captcha.
  • Se umny of the colors valores está faltando, ele será substituído pelo padrão.

Adding a Database¶

Para que a função captcha impeça alguém de enviar, you will need to add the information returned from create_captcha() to your database. on, when the data from the form is submitted by the user you will need to verify that the data exists in the database and has not expired.

Aqui está um protótipo de tabela:

CREATE TABLE captcha (
        captcha_id bigint(13) unsigned NOT NULL auto_increment,
        captcha_time int(10) unsigned NOT NULL,
        ip_address varchar(45) NOT NULL,
        word varchar(20) NOT NULL,
        PRIMARY KEY `captcha_id` (`captcha_id`),
        KEY `word` (`word`)
);

Here is an example of usage with a database. On the page where the CAPTCHA será mostrado que você terá algo assim:

$this->load->helper('captcha');
$vals = array(
        'img_path'      => './captcha/',
        'img_url'       => 'http://example.com/captcha/'
);

$cap = create_captcha($vals);
$data = array(
        'captcha_time'  => $cap['time'],
        'ip_address'    => $this->input->ip_address(),
        'word'          => $cap['word']
);

$query = $this->db->insert_string('captcha', $data);
$this->db->query($query);

echo 'Submit the word you see below:';
echo $cap['image'];
echo '<input type="text" name="captcha" value="" />';

Então, na página que aceita o envio, você terá algo como this:

// First, delete old captchas
$expiration = time() - 7200; // Two hour limit
$this->db->where('captcha_time < ', $expiration)
        ->delete('captcha');

// on see if a captcha exists:
$sql = 'SELECT COUNT(*) AS count FROM captcha WHERE word = ? AND ip_address = ? AND captcha_time > ?';
$binds = array($_POST['captcha'], $this->input->ip_address(), $expiration);
$query = $this->db->query($sql, $binds);
$row = $query->row();

if ($row->count == 0)
{
        echo 'You must submit the word that appears in the image.';
}

Funções Disponíveis¶

As seguintes funções estão disponíveis:

create_captcha([$data = ''[, $img_path = ''[, $img_url = ''[, $font_path = '']]]])¶
Parâmetros:
  • $data (array) - Matriz de dados para o CAPTCHA
  • $img_path (string) – Path to create the image in (DEPRECATED)
  • $img_url (string) - URL para a pasta de imagens CAPTCHA (DEPRECADO)
  • $font_path (string) - Caminho do servidor para a fonte (DEPRECADO)
Returns:

array(‘word’ => $word, ‘time’ => $now, ‘image’ => $img)
Return type:

array

Takes an array of information to generate the CAPTCHA as input and creates the image to your specifications, returning an array of associative data about the image.

array(
        'image' => IMAGE TAG
        'time'  => TIMESTAMP (in microtime)
        'word'  => CAPTCHA WORD
)

o image is the actual image tag:

<img src="http://example.com/captcha/12345.jpg" width="140" height="50" />

o time é o micro timestamp usado como o nome da imagem sem o file extension. It will be a number like this: 1139612155.3422

o word is the word that appears in the captcha image, which if not supplied to the function, will be a random string.

Nota

Usage of the $img_path, $img_url and $font_path parameters is DEPRECATED. Provide them in the $data array instead.

Helper CAPTCHA¶

o Helper CAPTCHA file contains functions that assist in creating Imagens de CAPTCHA.

  • Carregando este Helper
  • Usando o Helper CAPTCHA
    • Adicionando um Database
  • Funções Disponíveis

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('captcha');

Usando o Helper CAPTCHA¶

Uma vez carregado, você pode gerar um CAPTCHA assim:

$vals = array(
        'word'          => 'Random word',
        'img_path'      => './captcha/',
        'img_url'       => 'http://example.com/captcha/',
        'font_path'     => './path/to/fonts/texb.ttf',
        'img_width'     => '150',
        'img_height'    => 30,
        'expiration'    => 7200,
        'word_length'   => 8,
        'font_size'     => 16,
        'img_id'        => 'Imageid',
        'pool'          => '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ',

        // White background and border, black text and red grid
        'colors'        => array(
                'background' => array(255, 255, 255),
                'border' => array(255, 255, 255),
                'text' => array(0, 0, 0),
                'grid' => array(255, 40, 40)
        )
);

$cap = create_captcha($vals);
echo $cap['image'];
  • A função captcha requer a Library de imagens GD.
  • Only the img_path and img_url é requerido.
  • If a word não é fornecido, a função irá gerar um String ASCII. Você pode montar sua própria library de palavras que você pode desenhar aleatoriamente a partir de.
  • Se você não especificar um caminho para uma fonte TRUE TYPE, o GD feio nativo fonte será usada.
  • O diretório "captcha" deve ser gravável
  • o expiration (em segundos) significa quanto tempo uma imagem permanecerá na pasta captcha antes de ser apagada. O padrão é dois hours.
  • word_length o padrão é 8 pool O padrão é '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'
  • font_size o padrão é 16, a fonte GD nativa tem um limite de tamanho. Especifique uma fonte de "tipo verdadeiro" para tamanhos maiores.
  • o img_id será definido como o "id" da imagem captcha.
  • Se algum dos colors valores está faltando, ele será substituído pelo padrão.

Adicionando um Database¶

Para que a função captcha impeça alguém de enviar, você precisará adicionar as informações retornadas create_captcha() para o seu Database. Então, quando os dados do formulário são enviados por o usuário, você precisará verificar se os dados existem no Database e não expirou.

Aqui está um protótipo de tabela:

CREATE TABLE captcha (
        captcha_id bigint(13) unsigned NOT NULL auto_increment,
        captcha_time int(10) unsigned NOT NULL,
        ip_address varchar(45) NOT NULL,
        word varchar(20) NOT NULL,
        PRIMARY KEY `captcha_id` (`captcha_id`),
        KEY `word` (`word`)
);

Aqui está um exemplo de uso com um Database. Na página onde o CAPTCHA será mostrado que você terá algo assim:

$this->load->helper('captcha');
$vals = array(
        'img_path'      => './captcha/',
        'img_url'       => 'http://example.com/captcha/'
);

$cap = create_captcha($vals);
$data = array(
        'captcha_time'  => $cap['time'],
        'ip_address'    => $this->input->ip_address(),
        'word'          => $cap['word']
);

$query = $this->db->insert_string('captcha', $data);
$this->db->query($query);

echo 'Submit the word you see below:';
echo $cap['image'];
echo '<input type="text" name="captcha" value="" />';

Então, na página que aceita o envio, você terá algo como this:

// First, delete old captchas
$expiration = time() - 7200; // Two hour limit
$this->db->where('captcha_time < ', $expiration)
        ->delete('captcha');

// on see if a captcha exists:
$sql = 'SELECT COUNT(*) AS count FROM captcha WHERE word = ? AND ip_address = ? AND captcha_time > ?';
$binds = array($_POST['captcha'], $this->input->ip_address(), $expiration);
$query = $this->db->query($sql, $binds);
$row = $query->row();

if ($row->count == 0)
{
        echo 'You must submit the word that appears in the image.';
}
