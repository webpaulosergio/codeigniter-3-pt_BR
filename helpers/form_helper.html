Helper de Formulário¶

The Helper de Formulário file contains functions that assist in working with forms.

  • Loading this Helper
  • Escapando valores de campo
  • Funções Disponíveis

Loading this Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('form');

Escapando valores de campo¶

You may need to use HTML and characters such as quotes within your form elements. In order to do that safely, you’ll need to use common function html_escape().

Considere o seguinte exemplo:

$string = 'Here is a string containing "quoted" text.';

<input type="text" name="myfield" value="<?php echo $string; ?>" />

Como a string acima contém um conjunto de citações, ela fará com que o formulário to break. The html_escape() function converts HTML special characters so that it can be used safely:

<input type="text" name="myfield" value="<?php echo html_escape($string); ?>" />

Nota

If you use any of the form helper functions listed on this page, the form values will be automatically escaped, so there is no need para chamar esta função. Use-o somente se você estiver criando seu próprio form elements.

Funções Disponíveis¶

As seguintes funções estão disponíveis:

form_open([$action = ''[, $attributes = ''[, $hidden = array()]]])¶
Parâmetros:
  • $action (string) – Form action/target URI string
  • $attributes (array) - atributos HTML
  • $hidden (array) – An array of hidden fields’ definitions
Returns:

Uma tag de abertura de formulário HTML
Return type:

string

Cria uma tag de formulário de abertura com um URL base construído a partir de suas preferências de configuração. It will optionally let you add form attributes and hidden input fields, and will always add the accept-charset attribute based on the charset value in your arquivo de configuração.

The main benefit of using this tag rather than hard coding your own HTML is that Ele permite que seu site seja mais portável no caso de seus URLs serem alterados.

Here’s a simple example:

echo form_open('email/send');

O exemplo acima criaria um formulário que aponta para o URL base mais o Segmentos de URI "email/send", como este:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send">

Adicionando Atributos

Attributes can be added by passing an associative array to the second parameter, like this:

$attributes = array('class' => 'email', 'id' => 'myform');
echo form_open('email/send', $attributes);

Alternatively, you can specify the second parameter as a string:

echo form_open('email/send', 'class="email" id="myform"');

The above examples would create a form similar to this:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send" class="email" id="myform">

Adding Hidden Input Fields

Hidden fields can be added by passing an associative array to the terceiro parâmetro, assim:

$hidden = array('username' => 'Joe', 'member_id' => '234');
echo form_open('email/send', '', $hidden);

You can skip the second parameter by passing any falsy value to it.

The above example would create a form similar to this:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send">
        <input type="hidden" name="username" value="Joe" />
        <input type="hidden" name="member_id" value="234" />

form_hidden($name[, $value = ''])¶
Parâmetros:
  • $name (string) – Field name
  • $value (string) – Field value
Returns:

An HTML hidden input field tag
Return type:

string

Permite gerar campos de entrada ocultos. Você pode enviar um name/value string to create one field:

form_hidden('username', 'johndoe');
// Would produce: <input type="hidden" name="username" value="johndoe" />

… Ou você pode enviar uma matriz associativa para criar vários campos:

$data = array(
        'name'  => 'John Doe',
        'email' => 'john@example.com',
        'url'   => 'http://example.com'
);

echo form_hidden($data);

/*
        Would produce:
        <input type="hidden" name="name" value="John Doe" />
        <input type="hidden" name="email" value="john@example.com" />
        <input type="hidden" name="url" value="http://example.com" />
*/

Você também pode passar um array associativo para o campo de valor:

$data = array(
        'name'  => 'John Doe',
        'email' => 'john@example.com',
        'url'   => 'http://example.com'
);

echo form_hidden('my_array', $data);

/*
        Would produce:

        <input type="hidden" name="my_array[name]" value="John Doe" />
        <input type="hidden" name="my_array[email]" value="john@example.com" />
        <input type="hidden" name="my_array[url]" value="http://example.com" />
*/

Se você quiser criar campos de entrada ocultos com atributos extras:

$data = array(
        'type'  => 'hidden',
        'name'  => 'email',
        'id'    => 'hiddenemail',
        'value' => 'john@example.com',
        'class' => 'hiddenemail'
);

echo form_input($data);

/*
        Would produce:

        <input type="hidden" name="email" value="john@example.com" id="hiddenemail" class="hiddenemail" />
*/
form_input([$data = ''[, $value = ''[, $extra = '']]])¶
Parâmetros:
  • $data (array) – Field attributes data
  • $value (string) – Field value
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Returns:

Uma tag de campo de entrada de texto HTML
Return type:

string

Lets you generate a standard text input field. You can minimally pass the field name and value in the first and second parameter:

echo form_input('username', 'johndoe');

Or you can pass an associative array containing any data you wish your form to contain:

$data = array(
        'name'          => 'username',
        'id'            => 'username',
        'value'         => 'johndoe',
        'maxlength'     => '100',
        'size'          => '50',
        'style'         => 'width:50%'
);

echo form_input($data);

/*
        Would produce:

        <input type="text" name="username" value="johndoe" id="username" maxlength="100" size="50" style="width:50%"  />
*/

Se você quiser que seu formulário contenha alguns dados adicionais, como JavaScript, you can pass it as a string in the third parameter:

$js = 'onClick="some_function()"';
echo form_input('username', 'johndoe', $js);

Or you can pass it as an array:

$js = array('onClick' => 'some_function();');
echo form_input('username', 'johndoe', $js);

form_dropdown([$name = ''[, $options = array()[, $selected = array()[, $extra = '']]]])¶
Parâmetros:
  • $name (string) – Field name
  • $options (array) - Uma matriz associativa de opções a serem listadas
  • $selected (array) – List of fields to mark with the selected attribute
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Returns:

An HTML dropdown select field tag
Return type:

string

Lets you create a standard drop-down field. The first parameter will contain the name of the field, the second parameter will contain an associative array of options, and the third parameter will contain the value you wish to be selected. You can also pass an array of multiple items through the third parameter, and CodeIgniter will create a seleção múltipla para você.

Example:

$options = array(
        'small'         => 'Small Shirt',
        'med'           => 'Medium Shirt',
        'large'         => 'Large Shirt',
        'xlarge'        => 'Extra Large Shirt',
);

$shirts_on_sale = array('small', 'large');
echo form_dropdown('shirts', $options, 'large');

/*
        Would produce:

        <select name="shirts">
                <option value="small">Small Shirt</option>
                <option value="med">Medium  Shirt</option>
                <option value="large" selected="selected">Large Shirt</option>
                <option value="xlarge">Extra Large Shirt</option>
        </select>
*/

echo form_dropdown('shirts', $options, $shirts_on_sale);

/*
        Would produce:

        <select name="shirts" multiple="multiple">
                <option value="small" selected="selected">Small Shirt</option>
                <option value="med">Medium  Shirt</option>
                <option value="large" selected="selected">Large Shirt</option>
                <option value="xlarge">Extra Large Shirt</option>
        </select>
*/

If you would like the opening <select> to contain additional data, like um atributo id ou JavaScript, você pode passá-lo como uma string no quarto parameter:

$js = 'id="shirts" onChange="some_function();"';
echo form_dropdown('shirts', $options, 'large', $js);

Or you can pass it as an array:

$js = array(
        'id'       => 'shirts',
        'onChange' => 'some_function();'
);
echo form_dropdown('shirts', $options, 'large', $js);

If the array passed as $options is a multidimensional array, then form_dropdown() will produce an <optgroup> with the array key as the label.

form_multiselect([$name = ''[, $options = array()[, $selected = array()[, $extra = '']]]])¶
Parâmetros:
  • $name (string) – Field name
  • $options (array) - Uma matriz associativa de opções a serem listadas
  • $selected (array) – List of fields to mark with the selected attribute
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Returns:

Uma tag de campo multiseleta suspensa de HTML
Return type:

string

Permite criar um campo padrão de seleção múltipla. O primeiro parâmetro será contain the name of the field, the second parameter will contain an associative array of options, and the third parameter will contain the value or values you wish to be selected.

The parameter usage is identical to using form_dropdown() above, exceto, claro, que o nome do campo precisará usar o array POST sintaxe, por exemplo, foo [].

form_fieldset([$legend_text = ''[, $attributes = array()]])¶
Parâmetros:
  • $legend_text (string) – Text to put in the <legend> tag
  • $attributes (array) – Attributes to be set on the <fieldset> tag
Returns:

Uma tag de abertura de conjunto de campos HTML
Return type:

string

Lets you generate fieldset/legend fields.

Example:

echo form_fieldset('Address Information');
echo "<p>fieldset content here</p>\n";
echo form_fieldset_close();

/*
        Produces:

                <fieldset>
                        <legend>Address Information</legend>
                                <p>form content here</p>
                </fieldset>
*/

Similar to other functions, you can submit an associative array in the segundo parâmetro, se você preferir definir atributos adicionais:

$attributes = array(
        'id'    => 'address_info',
        'class' => 'address_info'
);

echo form_fieldset('Address Information', $attributes);
echo "<p>fieldset content here</p>\n";
echo form_fieldset_close();

/*
        Produces:

        <fieldset id="address_info" class="address_info">
                <legend>Address Information</legend>
                <p>form content here</p>
        </fieldset>
*/
form_fieldset_close([$extra = ''])¶
Parâmetros:
  • $extra (string) – Anything to append after the closing tag, as is
Returns:

An HTML fieldset closing tag
Return type:

string

Produces a closing </fieldset> tag. The only advantage to using this function is it permits you to pass data to it which will be added below a etiqueta. Por exemplo

$string = '</div></div>';
echo form_fieldset_close($string);
// Would produce: </fieldset></div></div>
form_checkbox([$data = ''[, $value = ''[, $checked = FALSE[, $extra = '']]]])¶
Parâmetros:
  • $data (array) – Field attributes data
  • $value (string) – Field value
  • $checked (bool) – Whether to mark the checkbox as being checked
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Returns:

An HTML checkbox input tag
Return type:

string

Lets you generate a checkbox field. Simple example:

echo form_checkbox('newsletter', 'accept', TRUE);
// Would produce:  <input type="checkbox" name="newsletter" value="accept" checked="checked" />

The third parameter contains a boolean TRUE/FALSE to determine whether a caixa deve ser verificada ou não.

Similar to the other form functions in this helper, you can also pass an array of attributes to the function:

$data = array(
        'name'          => 'newsletter',
        'id'            => 'newsletter',
        'value'         => 'accept',
        'checked'       => TRUE,
        'style'         => 'margin:10px'
);

echo form_checkbox($data);
// Would produce: <input type="checkbox" name="newsletter" id="newsletter" value="accept" checked="checked" style="margin:10px" />

Também como com outras funções, se você quiser que a tag contenha additional data like JavaScript, you can pass it as a string in the fourth parameter:

$js = 'onClick="some_function()"';
echo form_checkbox('newsletter', 'accept', TRUE, $js);

Or you can pass it as an array:

$js = array('onClick' => 'some_function();');
echo form_checkbox('newsletter', 'accept', TRUE, $js);

form_close([$extra = ''])¶
Parâmetros:
  • $extra (string) – Anything to append after the closing tag, as is
Returns:

An HTML form closing tag
Return type:

string

Produces a closing </form> tag. The only advantage to using this function is it permits you to pass data to it which will be added below a etiqueta. Por exemplo:

$string = '</div></div>';
echo form_close($string);
// Would produce:  </form> </div></div>
set_value($field[, $default = ''[, $html_escape = TRUE]])¶
Parâmetros:
  • $field (string) – Field name
  • $default (string) - Valor padrão
  • $html_escape (bool) – Whether to turn off HTML escaping of the value
Returns:

Field value
Return type:

string

Permits you to set the value of an input form or textarea. You must supply the field name via the first parameter of the function. The second (optional) parameter allows you to set a default value for the form. The third (optional) parameter allows you to turn off HTML escaping do valor, caso você precise usar esta função em combinação com i.e. form_input() and avoid double-escaping.

Example:

<input type="text" name="quantity" value="<?php echo set_value('quantity', '0'); ?>" size="50" />

O formulário acima mostrará "0" quando carregado pela primeira vez.

Nota

If you’ve loaded the Validação de Formulário Library and ter definido uma regra de validação para o nome do campo em uso com esse helper, ele será encaminhar a chamada para o Validação de Formulário Library’s own set_value() method. Otherwise, this function looks in $_POST for the field value.

set_select($field[, $value = ''[, $default = FALSE]])¶
Parâmetros:
  • $field (string) – Field name
  • $value (string) - Valor para verificar
  • $default (string) – Whether the value is also a default one
Returns:

‘selected’ attribute or an empty string
Return type:

string

If you use a <select> menu, this function permits you to display the menu item that was selected.

O primeiro parâmetro deve conter o nome do menu de seleção, o segundo parâmetro deve conter o valor de cada item e o terceiro (opcional) parameter lets you set an item as the default (use boolean TRUE/FALSE).

Example:

<select name="myselect">
        <option value="one" <?php echo  set_select('myselect', 'one', TRUE); ?> >One</option>
        <option value="two" <?php echo  set_select('myselect', 'two'); ?> >Two</option>
        <option value="three" <?php echo  set_select('myselect', 'three'); ?> >Three</option>
</select>
set_checkbox($field[, $value = ''[, $default = FALSE]])¶
Parâmetros:

  • $default (string) – Whether the value is also a default one

atributo 'checked' ou uma string vazia

string

echo validation_errors('<span class="error">', '</span>');

 */

Helper de Formulário¶

The Helper de Formulário file contains functions that assist in working with forms.

  • Carregando este Helper
  • Escapando valores de campo
  • Funções Disponíveis

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('form');

Escapando valores de campo¶

Você pode precisar usar HTML e caracteres como citações em seu formulário elementos. Para fazer isso com segurança, você precisará usar função comum html_escape().

Considere o seguinte exemplo:

$string = 'Here is a string containing "quoted" text.';

<input type="text" name="myfield" value="<?php echo $string; ?>" />

Como a string acima contém um conjunto de citações, ela fará com que o formulário quebrar. o html_escape() função converte especial em HTML caracteres para que possa ser usado com segurança:

<input type="text" name="myfield" value="<?php echo html_escape($string); ?>" />

Nota

Se você usar qualquer uma das funções Helperes de formulário listadas nesta página, os valores do formulário serão automaticamente ignorados, portanto não há necessidade para chamar esta função. Use-o somente se você estiver criando seu próprio elementos de formulário.

Funções Disponíveis¶

As seguintes funções estão disponíveis:

form_open([$action = ''[, $attributes = ''[, $hidden = array()]]])¶
Parâmetros:
  • $action (string) - String de URI de ação/destino de formulário
  • $attributes (array) - atributos HTML
  • $hidden (array) - Uma matriz de definições de campos ocultos
Retorna:

Uma tag de abertura de formulário HTML
Tipo de retorno:

string

Cria uma tag de formulário de abertura com um URL base construído a partir de suas preferências de configuração. Opcionalmente, você adicionará atributos de formulário e campos de entrada ocultos, e sempre adicionará o accept-charset atributo baseado no valor charset no seu arquivo de configuração.

A principal vantagem de usar essa tag em vez de codificar seu próprio HTML é Ele permite que seu site seja mais portável no caso de seus URLs serem alterados.

Aqui está um exemplo simples:

echo form_open('email/send');

O exemplo acima criaria um formulário que aponta para o URL base mais o Segmentos de URI "email/send", como este:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send">

Adicionando Atributos

Atributos podem ser adicionados passando um array associativo ao segundo parâmetro, assim:

$attributes = array('class' => 'email', 'id' => 'myform');
echo form_open('email/send', $attributes);

Alternativamente, você pode especificar o segundo parâmetro como uma string:

echo form_open('email/send', 'class="email" id="myform"');

Os exemplos acima criariam um formulário semelhante a este:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send" class="email" id="myform">

Adicionando campos de entrada ocultos

Campos ocultos podem ser adicionados passando um array associativo ao third parâmetro, assim:

$hidden = array('username' => 'Joe', 'member_id' => '234');
echo form_open('email/send', '', $hidden);

Você pode pular o segundo parâmetro passando qualquer valor falso para ele.

O exemplo acima criaria um formulário semelhante a este:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send">
        <input type="hidden" name="username" value="Joe" />
        <input type="hidden" name="member_id" value="234" />
form_open_multipart([$action = ''[, $attributes = array()[, $hidden = array()]]])¶
Parâmetros:
  • $action (string) - String de URI de ação/destino de formulário
  • $attributes (array) - atributos HTML
  • $hidden (array) - Uma matriz de definições de campos ocultos
Retorna:

Uma tag de abertura de formulários multipartes em HTML
Tipo de retorno:

string

Esta função é absolutamente idêntica a form_open() above, except that it adds a multipart attribute, which is necessary if you gostaria de usar o formulário para fazer upload de arquivos com.

form_hidden($name[, $value = ''])¶
Parâmetros:
  • $name (string) - Nome do campo
  • $value (string) - valor do campo
Retorna:

Uma tag de campo de entrada oculta em HTML
Tipo de retorno:

string

Permite gerar campos de entrada ocultos. Você pode enviar um string nome/valor para criar um campo:

form_hidden('username', 'johndoe');
// Would produce: <input type="hidden" name="username" value="johndoe" />

… Ou você pode enviar uma matriz associativa para criar vários campos:

$data = array(
        'name'  => 'John Doe',
        'email' => 'john@example.com',
        'url'   => 'http://example.com'
);

echo form_hidden($data);

/*
        Would produce:
        <input type="hidden" name="name" value="John Doe" />
        <input type="hidden" name="email" value="john@example.com" />
        <input type="hidden" name="url" value="http://example.com" />
*/

Você também pode passar um array associativo para o campo de valor:

$data = array(
        'name'  => 'John Doe',
        'email' => 'john@example.com',
        'url'   => 'http://example.com'
);

echo form_hidden('my_array', $data);

/*
        Would produce:

        <input type="hidden" name="my_array[name]" value="John Doe" />
        <input type="hidden" name="my_array[email]" value="john@example.com" />
        <input type="hidden" name="my_array[url]" value="http://example.com" />
*/

Se você quiser criar campos de entrada ocultos com atributos extras:

$data = array(
        'type'  => 'hidden',
        'name'  => 'email',
        'id'    => 'hiddenemail',
        'value' => 'john@example.com',
        'class' => 'hiddenemail'
);

echo form_input($data);

/*
        Would produce:

        <input type="hidden" name="email" value="john@example.com" id="hiddenemail" class="hiddenemail" />
*/
form_input([$data = ''[, $value = ''[, $extra = '']]])¶
Parâmetros:
  • $data (array) - Dados dos atributos do campo
  • $value (string) - valor do campo
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Retorna:

Uma tag de campo de entrada de texto HTML
Tipo de retorno:

string

Permite gerar um campo de entrada de texto padrão. Você pode passar minimamente o nome do campo e valor no primeiro e segundo parâmetro:

echo form_input('username', 'johndoe');

Ou você pode passar um array associativo contendo todos os dados que você deseja formulário para conter:

$data = array(
        'name'          => 'username',
        'id'            => 'username',
        'value'         => 'johndoe',
        'maxlength'     => '100',
        'size'          => '50',
        'style'         => 'width:50%'
);

echo form_input($data);

/*
        Would produce:

        <input type="text" name="username" value="johndoe" id="username" maxlength="100" size="50" style="width:50%"  />
*/

Se você quiser que seu formulário contenha alguns dados adicionais, como JavaScript, você pode passá-lo como uma string no terceiro parâmetro:

$js = 'onClick="some_function()"';
echo form_input('username', 'johndoe', $js);

Ou você pode passá-lo como um array:

$js = array('onClick' => 'some_function();');
echo form_input('username', 'johndoe', $js);
form_password([$data = ''[, $value = ''[, $extra = '']]])¶
Parâmetros:
  • $data (array) - Dados dos atributos do campo
  • $value (string) - valor do campo
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Retorna:

Uma tag de campo de entrada de senha HTML
Tipo de retorno:

string

Esta função é idêntica em todos os aspectos ao form_input() função acima, exceto que ele usa o tipo de entrada “senha”.

form_upload([$data = ''[, $value = ''[, $extra = '']]])¶
Parâmetros:
  • $data (array) - Dados dos atributos do campo
  • $value (string) - valor do campo
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Retorna:

Uma tag de campo de entrada de upload de arquivo HTML
Tipo de retorno:

string

Esta função é idêntica em todos os aspectos ao form_input() função acima, exceto que ele usa o tipo de entrada “arquivo”, permitindo que ser usado para fazer upload de arquivos.

form_textarea([$data = ''[, $value = ''[, $extra = '']]])¶
Parâmetros:
  • $data (array) - Dados dos atributos do campo
  • $value (string) - valor do campo
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Retorna:

Uma tag de textarea em HTML
Tipo de retorno:

string

Esta função é idêntica em todos os aspectos ao form_input() função acima, exceto que ele gera um tipo de “textarea”.

Nota

Instead of the maxlength and size attributes in the above example, you will instead specify rows and cols.

form_dropdown([$name = ''[, $options = array()[, $selected = array()[, $extra = '']]]])¶
Parâmetros:
  • $name (string) - Nome do campo
  • $options (array) - Uma matriz associativa de opções a serem listadas
  • $selected (array) – List of fields to mark with the selected attribute
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Retorna:

Uma lista suspensa de seleção de HTML
Tipo de retorno:

string

Permite criar um campo suspenso padrão. O primeiro parâmetro será contém o nome do campo, o segundo parâmetro conterá um matriz associativa de opções, e o terceiro parâmetro conterá o valor que você deseja ser selecionado. Você também pode passar uma matriz de múltiplos itens através do terceiro parâmetro, e o CodeIgniter irá criar um seleção múltipla para você.

Exemplo:

$options = array(
        'small'         => 'Small Shirt',
        'med'           => 'Medium Shirt',
        'large'         => 'Large Shirt',
        'xlarge'        => 'Extra Large Shirt',
);

$shirts_on_sale = array('small', 'large');
echo form_dropdown('shirts', $options, 'large');

/*
        Would produce:

        <select name="shirts">
                <option value="small">Small Shirt</option>
                <option value="med">Medium  Shirt</option>
                <option value="large" selected="selected">Large Shirt</option>
                <option value="xlarge">Extra Large Shirt</option>
        </select>
*/

echo form_dropdown('shirts', $options, $shirts_on_sale);

/*
        Would produce:

        <select name="shirts" multiple="multiple">
                <option value="small" selected="selected">Small Shirt</option>
                <option value="med">Medium  Shirt</option>
                <option value="large" selected="selected">Large Shirt</option>
                <option value="xlarge">Extra Large Shirt</option>
        </select>
*/

If you would like the opening <select> to contain additional data, like um atributo id ou JavaScript, você pode passá-lo como uma string no quarto parameter:

$js = 'id="shirts" onChange="some_function();"';
echo form_dropdown('shirts', $options, 'large', $js);

Ou você pode passá-lo como um array:

$js = array(
        'id'       => 'shirts',
        'onChange' => 'some_function();'
);
echo form_dropdown('shirts', $options, 'large', $js);

If the array passed as $options is a multidimensional array, then form_dropdown() will produce an <optgroup> with the array key as the label.

form_multiselect([$name = ''[, $options = array()[, $selected = array()[, $extra = '']]]])¶
Parâmetros:
  • $name (string) - Nome do campo
  • $options (array) - Uma matriz associativa de opções a serem listadas
  • $selected (array) – List of fields to mark with the selected attribute
  • $extra (mixed) - Atributos extras a serem adicionados à tag como uma matriz ou uma string literal
Retorna:

Uma tag de campo multiseleta suspensa de HTML
Tipo de retorno:

string

Permite criar um campo padrão de seleção múltipla. O primeiro parâmetro será contém o nome do campo, o segundo parâmetro conterá um matriz associativa de opções, e o terceiro parâmetro conterá o valor ou valores que você deseja selecionar.

O uso do parâmetro é idêntico ao uso form_dropdown() above, exceto, claro, que o nome do campo precisará usar o array POST sintaxe, por exemplo, foo [].

  • $legend_text (string) – Text to put in the <legend> tag
  • $attributes (array) – Attributes to be set on the <fieldset> tag
Retorna:

Uma tag de abertura de conjunto de campos HTML
Tipo de retorno:

string

Permite gerar campos fieldset/legenda.

echo form_fieldset('Address Information');

        Produces:

                <fieldset>
                        <legend>Address Information</legend>
                                <p>form content here</p>
                </fieldset>

$attributes = array(

echo form_fieldset('Address Information', $attributes);
echo "<p>fieldset content here</p>\n";

/*
        Produces:

        <fieldset id="address_info" class="address_info">
                <legend>Address Information</legend>
                <p>form content here</p>

echo form_checkbox('newsletter', 'accept', TRUE);

O terceiro parâmetro contém um booleano TRUE/FALSE para determinar se a caixa deve ser verificada ou não.

Semelhante às outras funções de formulário neste Helper, você também pode passar um matriz de atributos para a função:

$data = array(

        'value'         => 'accept',
        'checked'       => TRUE,
        'style'         => 'margin:10px'

echo form_checkbox($data);

$js = 'onClick="some_function()"';

$js = array('onClick' => 'some_function();');

set_value($field[, $default = ''[, $html_escape = TRUE]])¶
Parâmetros:
  • $field (string) - Nome do campo

Retorna:

Valor do campo
Tipo de retorno:

string

Permite que você defina o valor de um formulário de entrada ou uma área de texto. Você deve forneça o nome do campo através do primeiro parâmetro da função. o segundo parâmetro (opcional) permite que você defina um valor padrão para o Formato. O terceiro parâmetro (opcional) permite desativar o escape de HTML do valor, caso você precise usar esta função em combinação com i.e. form_input() e evite o escape duplo.

Exemplo:

<input type="text" name="quantity" value="<?php echo set_value('quantity', '0'); ?>" size="50" />

O formulário acima mostrará "0" quando carregado pela primeira vez.

Se você carregou o Validação de Formulário Library and ter definido uma regra de validação para o nome do campo em uso com esse helper, ele será encaminhar a chamada para o Validação de Formulário Library’s own set_value() method. Otherwise, this function looks in $_POST for the valor do campo.

Helper de Formulário¶

The Helper de Formulário file contains functions that assist in working with forms.

  • Carregando este Helper
  • Escapando valores de campo
  • Funções Disponíveis

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('form');

Escapando valores de campo¶

Você pode precisar usar HTML e caracteres como citações em seu formulário elementos. Para fazer isso com segurança, você precisará usar função comum html_escape().

Considere o seguinte exemplo:

$string = 'Here is a string containing "quoted" text.';

<input type="text" name="myfield" value="<?php echo $string; ?>" />

Como a string acima contém um conjunto de citações, ela fará com que o formulário quebrar. o html_escape() função converte especial em HTML caracteres para que possa ser usado com segurança:

<input type="text" name="myfield" value="<?php echo html_escape($string); ?>" />

Nota

Se você usar qualquer uma das funções Helperes de formulário listadas nesta página, os valores do formulário serão automaticamente ignorados, portanto não há necessidade para chamar esta função. Use-o somente se você estiver criando seu próprio elementos de formulário.

Funções Disponíveis¶

As seguintes funções estão disponíveis:

form_open([$action = ''[, $attributes = ''[, $hidden = array()]]])¶
Parâmetros:
  • $action (string) - String de URI de ação/destino de formulário
  • $attributes (array) - atributos HTML
  • $hidden (array) - Uma matriz de definições de campos ocultos
Retorna:

Uma tag de abertura de formulário HTML
Tipo de retorno:

string

Cria uma tag de formulário de abertura com um URL base construído a partir de suas preferências de configuração. Opcionalmente, você adicionará atributos de formulário e campos de entrada ocultos, e sempre adicionará o accept-charset atributo baseado no valor charset no seu arquivo de configuração.

A principal vantagem de usar essa tag em vez de codificar seu próprio HTML é Ele permite que seu site seja mais portável no caso de seus URLs serem alterados.

Aqui está um exemplo simples:

echo form_open('email/send');

O exemplo acima criaria um formulário que aponta para o URL base mais o Segmentos de URI "email/send", como este:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send">

Adicionando Atributos

Atributos podem ser adicionados passando um array associativo ao segundo parâmetro, assim:

$attributes = array('class' => 'email', 'id' => 'myform');
echo form_open('email/send', $attributes);

Alternativamente, você pode especificar o segundo parâmetro como uma string:

echo form_open('email/send', 'class="email" id="myform"');

Os exemplos acima criariam um formulário semelhante a este:

<form method="post" accept-charset="utf-8" action="http://example.com/index.php/email/send" class="email" id="myform">

Adicionando campos de entrada ocultos

Campos ocultos podem ser adicionados passando um array associativo ao third parâmetro, assim:

$hidden = array('username' => 'Joe', 'member_id' => '234');
echo form_open('email/send', '', $hidden);

Você pode pular o segundo parâmetro passando qualquer valor falso para ele.

form_dropdown([$name = ''[, $options = array()[, $selected = array()[, $extra = '']]]])¶
Parâmetros:

Permite criar um campo suspenso padrão.         'large'         => 'Large Shirt',
        'xlarge'        => 'Extra Large Shirt',
);

        Would produce:

        <select name="shirts">
                <option value="small">Small Shirt</option>
                <option value="med">Medium  Shirt</option>

                <option value="xlarge">Extra Large Shirt</option>
        </select>
*/

/*
        Would produce:

        <select name="shirts" multiple="multiple">
                <option value="small" selected="selected">Small Shirt</option>
                <option value="med">Medium  Shirt</option>
                <option value="large" selected="selected">Large Shirt</option>
                <option value="xlarge">Extra Large Shirt</option>
        </select>
*/

Se você quiser que a abertura <select> contenha dados adicionais, como um atributo id ou JavaScript, você pode passá-lo como uma string no quarto parameter:

Ou você pode passá-lo como um array:

$js = array(
        'id'       => 'shirts',
        'onChange' => 'some_function();'
);
echo form_dropdown('shirts', $options, 'large', $js);
