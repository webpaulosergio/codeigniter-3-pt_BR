Auxiliar de Texto¶

The Auxiliar de Texto file contains functions that assist in working with text.

  • Loading this Helper
  • Funções Disponíveis

Loading this Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('text');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

word_limiter($str[, $limit = 100[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) – Input string
  • $limit (int) – Limit
  • $end_char (string) – End character (usually an ellipsis)
Returns:

Word-limited string
Return type:

string

Truncates a string to the number of words specified. Example:

$string = "Here is a nice text string consisting of eleven words.";
$string = word_limiter($string, 4);
// Returns:  Here is a nice

The third parameter is an optional suffix added to the string. By Por padrão, ele adiciona reticências.

character_limiter($str[, $n = 500[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) – Input string
  • $n (int) – Number of characters
  • $end_char (string) – End character (usually an ellipsis)
Returns:

Cadeia limitada por caracteres
Return type:

string

Truncates a string to the number of characters specified. It maintains the integrity of words so the character count may be slightly more or less than what you specify.

Example:

$string = "Here is a nice text string consisting of eleven words.";
$string = character_limiter($string, 20);
// Returns:  Here is a nice text string

The third parameter is an optional suffix added to the string, if undeclared this helper uses an ellipsis.

Nota

If you need to truncate to an exact number of characters please Veja o ellipsize() função abaixo.

ascii_to_entities($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

Uma string com valores ASCII convertidos em entidades
Return type:

string

Converte valores ASCII em entidades de caracteres, incluindo alto ASCII e MS Word characters that can cause problems when used in a web page, so that eles podem ser exibidos de forma consistente, independentemente das configurações do navegador ou armazenados de forma confiável em um database. Há alguma dependência em seu servidor conjuntos de caracteres suportados, por isso pode não ser 100% confiável em todos os casos, mas na maior parte, deve identificar corretamente os caracteres fora o alcance normal (como caracteres acentuados).

Example:

$string = ascii_to_entities($string);
convert_accented_characters($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

A string with accented characters converted
Return type:

string

Transliterates high ASCII characters to low ASCII equivalents. Useful when non-English characters need to be used where only standard ASCII characters are safely used, for instance, in URLs.

Example:

$string = convert_accented_characters($string);

Nota

This function uses a companion config file application/config/foreign_chars.php to define the to and de matriz para transliteração.

word_censor($str, $censored[, $replacement = ''])¶
Parâmetros:
  • $str (string) – Input string
  • $censored (array) – List of bad words to censor
  • $replacement (string) - O que substituir palavras ruins com
Returns:

Cadeia censurada
Return type:

string

Enables you to censor words within a text string. The first parameter will contain the original string. The second will contain an array of palavras que você não permite. O terceiro parâmetro (opcional) pode conter a replacement value for the words. If not specified they are replaced com sinais de libra: ####.

Example:

$disallowed = array('darn', 'shucks', 'golly', 'phooey');
$string = word_censor($string, $disallowed, 'Beep!');
highlight_code($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

String with code highlighted via HTML
Return type:

string

Coloriza uma string de código (PHP, HTML, etc.). Exemplo:

$string = highlight_code($string);

The function uses PHP’s highlight_string() function, so the As cores usadas são aquelas especificadas no seu arquivo php.ini.

highlight_phrase($str, $phrase[, $tag_open = '<mark>'[, $tag_close = '</mark>']])¶
Parâmetros:
  • $str (string) – Input string
  • $phrase (string) – Phrase to highlight
  • $tag_open (string) – Opening tag used for the highlight
  • $tag_close (string) – Closing tag for the highlight
Returns:

String com uma frase destacada via HTML
Return type:

string

Will highlight a phrase within a text string. The first parameter will contain the original string, the second will contain the phrase you wish to highlight. The third and fourth parameters will contain the Abrindo/fechando tags HTML que você gostaria que a frase envolto em.

Example:

$string = "Here is a nice text string about nothing in particular.";
echo highlight_phrase($string, "nice text", '<span style="color:#990000;">', '</span>');

The above code prints:

Here is a <span style="color:#990000;">nice text</span> string about nothing in particular.

Nota

This function used to use the <strong> tag by default. Older browsers might not support the new HTML5 mark tag, so it is recommended that you insira o seguinte código CSS na sua folha de estilo, se precisar de suporte tais navegadores:

mark {
        background: #ff0;
        color: #000;
};
word_wrap($str[, $charlim = 76])¶
Parâmetros:
  • $str (string) – Input string
  • $charlim (int) – Character limit
Returns:

Corda embrulhada no Word
Return type:

string

Wraps text at the specified character count while maintaining palavras completas.

Example:

$string = "Here is a simple string of text that will help us demonstrate this function.";
echo word_wrap($string, 25);

// Would produce:
// Here is a simple string
// of text that will help us
// demonstrate this
// function.
ellipsize($str, $max_length[, $position = 1[, $ellipsis = '&hellip;']])¶
Parâmetros:
  • $str (string) – Input string
  • $max_length (int) – String length limit
  • $position (mixed) - Posição para dividir em (int ou float)
  • $ellipsis (string) – What to use as the ellipsis character
Returns:

Corda Elipsizada
Return type:

string

Esta função irá remover tags de uma string, dividi-la em um comprimento máximo e insira uma elipse.

O primeiro parâmetro é a string a ser reticiada, o segundo é o número of characters in the final string. The third parameter is where in the string the ellipsis should appear from 0 - 1, left to right. For exemplo. um valor de 1 colocará as reticências à direita do string, .5 in the middle, and 0 at the left.

Um parâmetro opcional é o tipo de reticências. Por padrão, &hellip; will be inserted.

Example:

$str = 'this_string_is_entirely_too_long_and_might_break_my_design.jpg';
echo ellipsize($str, 32, .5);

Produz:

this_string_is_e&hellip;ak_my_design.jpg

$this->load->helper('text');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

word_limiter($str[, $limit = 100[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) - string de entrada
  • $limit (int) - limite
  • $end_char (string) - caractere final (geralmente uma elipse)
Retorna:

Cadeia limitada por palavras
Tipo de retorno:

string

Truncates a string to the number of words specified. Exemplo:

$string = "Here is a nice text string consisting of eleven words.";

// Retorna:  Here is a nice

O terceiro parâmetro é um sufixo opcional adicionado à string. Por Por padrão, ele adiciona reticências.

character_limiter($str[, $n = 500[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) - string de entrada
  • $n (int) - Número de caracteres
  • $end_char (string) - caractere final (geralmente uma elipse)
Retorna:

Cadeia limitada por caracteres
Tipo de retorno:

string

Truncates a string to the number of characters specified. It mantém a integridade das palavras para que a contagem de caracteres possa ser levemente mais ou menos do que você especifica.

$string = "Here is a nice text string consisting of eleven words.";
$string = character_limiter($string, 20);

O terceiro parâmetro é um sufixo opcional adicionado à string, se não declarado, esse Helper usa reticências.

Nota

Se você precisar truncar um número exato de caracteres, por favor Veja o ellipsize() função abaixo.

ascii_to_entities($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Uma string com valores ASCII convertidos em entidades
Tipo de retorno:

string

Converte valores ASCII em entidades de caracteres, incluindo alto ASCII e MS Caracteres do Word que podem causar problemas quando usados em uma página da Web, eles podem ser exibidos de forma consistente, independentemente das configurações do navegador ou armazenados de forma confiável em um database. Há alguma dependência em seu servidor conjuntos de caracteres suportados, por isso pode não ser 100% confiável em todos os casos, mas na maior parte, deve identificar corretamente os caracteres fora o alcance normal (como caracteres acentuados).

Exemplo:

$string = ascii_to_entities($string);
convert_accented_characters($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Uma string com caracteres acentuados convertidos
Tipo de retorno:

string

Translitera caracteres ASCII altos para baixos equivalentes ASCII. Útil quando caracteres não ingleses precisam ser usados onde somente o ASCII padrão caracteres são usados com segurança, por exemplo, em URLs.

Exemplo:

$string = convert_accented_characters($string);

Nota

Esta função usa um arquivo de configuração complementar application/config/foreign_chars.php para definir o para e de matriz para transliteração.

word_censor($str, $censored[, $replacement = ''])¶
Parâmetros:
  • $str (string) - string de entrada
  • $censored (array) - Lista de palavrões para censurar
  • $replacement (string) - O que substituir palavras ruins com
Retorna:

Cadeia censurada

Helper de Texto¶

The Helper de Texto file contains functions that assist in working with text.

  • Carregando este Helper
  • Funções Disponíveis

Carregando este Helper¶

Este helper é carregado usando o seguinte código:

$this->load->helper('text');

Funções Disponíveis¶

As seguintes funções estão disponíveis:

word_limiter($str[, $limit = 100[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) - string de entrada
  • $limit (int) – Limit
  • $end_char (string) - caractere final (geralmente uma elipse)
Retorna:

Cadeia limitada por palavras
Tipo de retorno:

string

Truncates a string to the number of words specified. Exemplo:

$string = "Here is a nice text string consisting of eleven words.";
$string = word_limiter($string, 4);
// Retorna:  Here is a nice

O terceiro parâmetro é um sufixo opcional adicionado à string. Por Por padrão, ele adiciona reticências.

character_limiter($str[, $n = 500[, $end_char = '&#8230;']])¶
Parâmetros:
  • $str (string) - string de entrada
  • $n (int) - Número de caracteres
  • $end_char (string) - caractere final (geralmente uma elipse)
Retorna:

Cadeia limitada por caracteres
Tipo de retorno:

string

Truncates a string to the number of characters specified. It mantém a integridade das palavras para que a contagem de caracteres possa ser levemente mais ou menos do que você especifica.

Exemplo:

$string = "Here is a nice text string consisting of eleven words.";
$string = character_limiter($string, 20);
// Retorna:  Here is a nice text string

O terceiro parâmetro é um sufixo opcional adicionado à string, se não declarado, esse Helper usa reticências.

Nota

Se você precisar truncar um número exato de caracteres, por favor see the ellipsize() função abaixo.

ascii_to_entities($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Uma string com valores ASCII convertidos em entidades
Tipo de retorno:

string

Converte valores ASCII em entidades de caracteres, incluindo alto ASCII e MS Caracteres do Word que podem causar problemas quando usados em uma página da Web, eles podem ser exibidos de forma consistente, independentemente das configurações do navegador ou armazenados de forma confiável em um database. Há alguma dependência em seu servidor conjuntos de caracteres suportados, por isso pode não ser 100% confiável em todos os casos, mas na maior parte, deve identificar corretamente os caracteres fora o alcance normal (como caracteres acentuados).

Exemplo:

$string = ascii_to_entities($string);
convert_accented_characters($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Uma string com caracteres acentuados convertidos
Tipo de retorno:

string

Translitera caracteres ASCII altos para baixos equivalentes ASCII. Útil quando caracteres não ingleses precisam ser usados onde somente o ASCII padrão caracteres são usados com segurança, por exemplo, em URLs.

Exemplo:

$string = convert_accented_characters($string);

Nota

Esta função usa um arquivo de configuração complementar application/config/foreign_chars.php para definir o para e de matriz para transliteração.
