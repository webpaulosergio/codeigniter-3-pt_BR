# Documentação do CodeIgniter 3. em Português (pt_BR)


Tradução da documentação do CodeIgniter 3.
Alguns textos podem não estar totalmente traduzidor e a tradução pode estar imprecisa

A tradução foi criada automaticamente. Utilizamos um spider para mapear os textos das páginas em inglês da documentação, ignorando claro exemplos e códigos, também utilizamos o Google Translate API para gerar as traduções.

##Trabalhando
Estamos trabalhando manualmente em cada página, para deixar a tradução mais precisa, mas isso levará um tempo ;)

##Colaboração
Gostaria de colaborar conosco na tradução?
1. Realize um 'fork' deste repositório
2. Crie um arquivo 'changelog.md' com quais páginas e conteúdos você alterou
3. Envie um 'pull request' quando terminar
