Classe de codificação zip¶

CodeIgniter’s Classe de codificação zip permits you to create Zip archives. Archives can be downloaded to your desktop or saved to a directory.

  • Using the Classe de codificação zip
    • Inicializando a classe
    • Exemplo de uso
  • Class Reference

Using the Classe de codificação zip¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe Zip é inicializada em your controller using the $this->load->library function:

$this->load->library('zip');

Once loaded, the Zip library object will be available using:

$this->zip

Exemplo de uso¶

Este exemplo demonstra como compactar um arquivo, salvá-lo em uma pasta seu servidor e baixe-o para a sua área de trabalho.

$name = 'mydata1.txt';
$data = 'A Data String!';

$this->zip->add_data($name, $data);

// Write the zip file to a folder on your server. Name it "my_backup.zip"
$this->zip->archive('/path/to/directory/my_backup.zip');

// Download the file to your desktop. Name it "my_backup.zip"
$this->zip->download('my_backup.zip');

Class Reference¶

class CI_Zip¶
$compression_level = 2

O nível de compactação a ser usado.

Pode variar de 0 a 9, com 9 sendo a mais alta e 0 efetivamente desativando a compactação:

$this->zip->compression_level = 0;
add_data($filepath[, $data = NULL])¶
Parâmetros:
  • $filepath (mixed) – A single file path or an array of file => data pairs
  • $data (array) - Conteúdo do arquivo (ignorado se $filepath é um array)
Return type:

void

Adiciona dados ao arquivo Zip. Pode trabalhar tanto no modo de arquivos simples como múltiplos.

When adding a single file, the first parameter must contain the name you would like given to the file and the second must contain the file contents:

$name = 'mydata1.txt';
$data = 'A Data String!';
$this->zip->add_data($name, $data);

$name = 'mydata2.txt';
$data = 'Another Data String!';
$this->zip->add_data($name, $data);

When adding multiple files, the first parameter must contain file => contents pairs and the second parameter is ignored:

$data = array(
        'mydata1.txt' => 'A Data String!',
        'mydata2.txt' => 'Another Data String!'
);

$this->zip->add_data($data);

If you would like your compressed data organized into sub-directories, simply include the path as part of the filename(s):

$name = 'personal/my_bio.txt';
$data = 'I was born in an elevator...';

$this->zip->add_data($name, $data);

O exemplo acima colocará my_bio.txt dentro de uma pasta chamada pessoal.

add_dir($directory)¶
Parâmetros:
  • $directory (mixed) – Directory name string or an array of multiple directories
Return type:

void

Permite que você adicione um diretório. Geralmente este método é desnecessário, já que você pode colocar your data into directories when using $this->zip->add_data(), but if you would like to create an empty directory you can do so:

$this->zip->add_dir('myfolder'); // Creates a directory called "myfolder"
read_file($path[, $archive_filepath = FALSE])¶
Parâmetros:
  • $path (string) – Path to file
  • $archive_filepath (mixed) – New file name/path (string) or (boolean) whether to maintain the original filepath
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Permits you to compress a file that already exists somewhere on your server. Forneça um caminho de arquivo e a classe zip irá lê-lo e adicioná-lo ao arquivo:

$path = '/path/to/photo.jpg';

$this->zip->read_file($path);

// Download the file to your desktop. Name it "my_backup.zip"
$this->zip->download('my_backup.zip');

If you would like the Zip archive to maintain the directory structure of o arquivo nele, passa TRUE (booleano) no segundo parâmetro. Exemplo:

$path = '/path/to/photo.jpg';

$this->zip->read_file($path, TRUE);

// Download the file to your desktop. Name it "my_backup.zip"
$this->zip->download('my_backup.zip');

In the above example, photo.jpg will be placed into the path/to/ directory.

Você também pode especificar um novo nome (caminho incluído) para o arquivo adicionado em tempo real:

$path = '/path/to/photo.jpg';
$new_path = '/new/path/some_photo.jpg';

$this->zip->read_file($path, $new_path);

// Download ZIP archive containing /new/path/some_photo.jpg
$this->zip->download('my_archive.zip');
read_dir($path[, $preserve_filepath = TRUE[, $root_path = NULL]])¶
Parâmetros:
  • $path (string) – Path to directory
  • $preserve_filepath (bool) – Whether to maintain the original path
  • $root_path (string) – Part of the path to exclude from the archive directory
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Permits you to compress a directory (and its contents) that already exists somewhere on your server. Supply a path to the directory and the zip class will recursively read and recreate it as a Zip archive. Todos os arquivos contidos no caminho fornecido serão codificados, assim como qualquer subdiretório contido nele. Exemplo:

$path = '/path/to/your/directory/';

$this->zip->read_dir($path);

// Download the file to your desktop. Name it "my_backup.zip"
$this->zip->download('my_backup.zip');

By default the Zip archive will place all directories listed in the first parameter inside the zip. If you want the tree preceding the target directory to be ignored, you can pass FALSE (boolean) in the second parameter. Example:

$path = '/path/to/your/directory/';

$this->zip->read_dir($path, FALSE);

This will create a ZIP with a directory named “directory” inside, then all sub-directories stored correctly inside that, but will not include the /path/to/your part of the path.

archive($filepath)¶
Parâmetros:
  • $filepath (string) – Path to target zip archive
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Writes the Zip-encoded file to a directory on your server. Submit a valid server path ending in the file name. Make sure the directory is writable (755 is usually OK). Example:

$this->zip->archive('/path/to/folder/myarchive.zip'); // Creates a file named myarchive.zip
download($filename = 'backup.zip')¶
Parâmetros:
  • $filename (string) – Archive file name
Return type:

void

Causes the Zip file to be downloaded from your server. You must pass the name you would like the zip file called. Example:

$this->zip->download('latest_stuff.zip'); // File will be named "latest_stuff.zip"

Nota

Do not display any data in the controller in which you call this method since it sends various server headers that cause the download para acontecer e o arquivo a ser tratado como binário.

get_zip()¶
Returns:Zip file content
Return type:string

Retorna os dados do arquivo compactados com zip. Geralmente você não precisará deste método a menos que você want to do something unique with the data. Example:

$name = 'my_bio.txt';
$data = 'I was born in an elevator...';

$this->zip->add_data($name, $data);

$zip_file = $this->zip->get_zip();
clear_data()¶
Return type:void

A classe Zip armazena em cache seus dados zip para que não precise recompilar o arquivo Zip for each method you use above. If, however, you need to create multiple Zip archives, each with different data, you can clear the cache between calls. Example:

$name = 'my_bio.txt';
$data = 'I was born in an elevator...';

$this->zip->add_data($name, $data);
$zip_file = $this->zip->get_zip();

$this->zip->clear_data();

$name = 'photo.jpg';
$this->zip->read_file("/path/to/photo.jpg"); // Read the file's contents

$this->zip->download('myphotos.zip');

Classe de codificação zip¶

CodeIgniter’s Classe de codificação zip permits you to create Zip archives. Os arquivos podem ser baixados na sua área de trabalho ou salvos em um diretório.

  • Using the Classe de codificação zip
    • Inicializando a classe
    • Exemplo de uso

Using the Classe de codificação zip¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe Zip é inicializada em seu Controller usando a função $this-> load-> library:

$this->load->library('zip');

Uma vez carregado, o objeto da Library Zip estará disponível usando:

$this->zip

Exemplo de uso¶

Este exemplo demonstra como compactar um arquivo, salvá-lo em uma pasta seu servidor e baixe-o para a sua área de trabalho.

$name = 'mydata1.txt';
$data = 'A Data String!';

$this->zip->add_data($name, $data);

// Write the zip file to a folder on your server. Name it "my_backup.zip"
$this->zip->archive('/path/to/directory/my_backup.zip');

// Download the file to your desktop. Name it "my_backup.zip"
$this->zip->download('my_backup.zip');

Referência de Classe¶

class CI_Zip¶
$compression_level = 2

O nível de compactação a ser usado.

Pode variar de 0 a 9, com 9 sendo a mais alta e 0 efetivamente desativando a compactação:

$this->zip->compression_level = 0;
add_data($filepath[, $data = NULL])¶
Parâmetros:
  • $filepath (mixed) - Um único caminho de arquivo ou uma matriz de arquivo => pares de dados
  • $data (array) - Conteúdo do arquivo (ignorado se $filepath é um array)
Tipo de retorno:

void

Adiciona dados ao arquivo Zip. Pode trabalhar tanto no modo de arquivos simples como múltiplos.

Ao adicionar um único arquivo, o primeiro parâmetro deve conter o nome que você como dado ao arquivo e o segundo deve conter o conteúdo do arquivo:

$name = 'mydata1.txt';
$data = 'A Data String!';
$this->zip->add_data($name, $data);

$name = 'mydata2.txt';
$data = 'Another Data String!';
$this->zip->add_data($name, $data);

When adding multiple files, the first parameter must contain file => contents pairs e o segundo parâmetro é ignorado:

$data = array(
        'mydata1.txt' => 'A Data String!',
        'mydata2.txt' => 'Another Data String!'
);

$this->zip->add_data($data);

Se você quiser que seus dados compactados sejam organizados em subdiretórios, simplesmente inclua o caminho como parte do (s) nome (s) de arquivo (s):

$name = 'personal/my_bio.txt';
$data = 'I was born in an elevator...';

$this->zip->add_data($name, $data);

O exemplo acima colocará my_bio.txt dentro de uma pasta chamada pessoal.

add_dir($directory)¶
Parâmetros:
  • $directory (mixed) - Cadeia de nome de diretório ou uma matriz de vários diretórios
Tipo de retorno:

void

Classe de codificação zip¶

CodeIgniter’s Classe de codificação zip permits you to create Zip archives. Os arquivos podem ser baixados na sua área de trabalho ou salvos em um diretório.

  • Using the Classe de codificação zip
    • Inicializando a classe
    • Exemplo de uso
  • Referência de Classe

Using the Classe de codificação zip¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe Zip é inicializada em seu Controller usando a função $this->load->library function:

$this->load->library('zip');

Uma vez carregado, o objeto da Library Zip estará disponível usando:

$this->zip

Exemplo de uso¶

Este exemplo demonstra como compactar um arquivo, salvá-lo em uma pasta seu servidor e baixe-o para a sua área de trabalho.

$name = 'mydata1.txt';
$data = 'A Data String!';

$this->zip->add_data($name, $data);

// Write the zip file to a folder on your server. Name it "my_backup.zip"
$this->zip->archive('/path/to/directory/my_backup.zip');

// Download the file to your desktop. Name it "my_backup.zip"
$this->zip->download('my_backup.zip');

Referência de Classe¶

class CI_Zip¶
$compression_level = 2

O nível de compactação a ser usado.

Pode variar de 0 a 9, com 9 sendo a mais alta e 0 efetivamente desativando a compactação:

$this->zip->compression_level = 0;
add_data($filepath[, $data = NULL])¶
Parâmetros:
  • $filepath (mixed) - Um único caminho de arquivo ou uma matriz de arquivo => pares de dados
  • $data (array) - Conteúdo do arquivo (ignorado se $filepath é um array)
Tipo de retorno:

void

Adiciona dados ao arquivo Zip. Pode trabalhar tanto no modo de arquivos simples como múltiplos.

Ao adicionar um único arquivo, o primeiro parâmetro deve conter o nome que você como dado ao arquivo e o segundo deve conter o conteúdo do arquivo:

$name = 'mydata1.txt';
$data = 'A Data String!';
$this->zip->add_data($name, $data);

$name = 'mydata2.txt';
$data = 'Another Data String!';
$this->zip->add_data($name, $data);

When adding multiple files, the first parameter must contain file => contents pairs e o segundo parâmetro é ignorado:

$data = array(
        'mydata1.txt' => 'A Data String!',
        'mydata2.txt' => 'Another Data String!'
);

$this->zip->add_data($data);

Se você quiser que seus dados compactados sejam organizados em subdiretórios, simplesmente inclua o caminho como parte do (s) nome (s) de arquivo (s):

$name = 'personal/my_bio.txt';
$data = 'I was born in an elevator...';

$this->zip->add_data($name, $data);

O exemplo acima colocará my_bio.txt dentro de uma pasta chamada pessoal.
