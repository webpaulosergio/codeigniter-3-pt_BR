Classe FTP¶

CodeIgniter’s Classe FTP permits files to be transferred to a remote servidor. Arquivos remotos também podem ser movidos, renomeados e excluídos. O FTP classe também inclui uma função de “espelhamento” que permite que um diretório a ser recriado remotamente via FTP.

Nota

Os protocolos SFTP e SSL FTP não são suportados, somente padrão FTP.

    • Inicializando a classe

  • Class Reference

Working with the Classe FTP¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe FTP é inicializada em your controller using the $this->load->library function:

$this->load->library('ftp');

Once loaded, the FTP object will be available using: $this->ftp

Exemplos de uso¶

Neste exemplo, uma conexão é aberta ao servidor FTP e uma conexão local file is read and uploaded in ASCII mode. The file permissions are set to 755.

$this->load->library('ftp');

$config['hostname'] = 'ftp.example.com';
$config['username'] = 'your-username';
$config['password'] = 'your-password';
$config['debug']        = TRUE;

$this->ftp->connect($config);

$this->ftp->upload('/local/path/to/myfile.html', '/public_html/myfile.html', 'ascii', 0775);

$this->ftp->close();

In this example a list of files is retrieved from the server.

$this->load->library('ftp');

$config['hostname'] = 'ftp.example.com';
$config['username'] = 'your-username';
$config['password'] = 'your-password';
$config['debug']        = TRUE;

$this->ftp->connect($config);

$list = $this->ftp->list_files('/public_html/');

print_r($list);

$this->ftp->close();

In this example a local directory is mirrored on the server.

$this->load->library('ftp');

$config['hostname'] = 'ftp.example.com';
$config['username'] = 'your-username';
$config['password'] = 'your-password';
$config['debug']        = TRUE;

$this->ftp->connect($config);

$this->ftp->mirror('/path/to/myfolder/', '/public_html/myfolder/');

$this->ftp->close();

Class Reference¶

class CI_FTP¶
connect([$config = array()])¶
Parâmetros:
  • $config (array) – Connection values
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Connects and logs into to the FTP server. Connection preferences are set by passing an array to the function, or you can store them in a config file.

Here is an example showing how you set preferences manually:

$this->load->library('ftp');

$config['hostname'] = 'ftp.example.com';
$config['username'] = 'your-username';
$config['password'] = 'your-password';
$config['port']     = 21;
$config['passive']  = FALSE;
$config['debug']    = TRUE;

$this->ftp->connect($config);

Setting FTP Preferences in a Config File

If you prefer you can store your FTP preferences in a config file. Simplesmente crie um novo arquivo chamado ftp.php, adicione o array $config no that file. Then save the file at application/config/ftp.php and it será usado automaticamente.

Available connection options

Nome da opção Default value Descrição
hostname n/a FTP hostname (usually something like: ftp.example.com)
username n/a FTP username
password n/a FTP password
port 21 Número da porta do servidor FTP
debug FALSE TRUE/FALSE (boolean): Whether to enable debugging to display error messages
passive TRUE TRUE/FALSE (booleano): se deve usar o modo passivo
upload($locpath, $rempath[, $mode = 'auto'[, $permissions = NULL]])¶
Parâmetros:
  • $locpath (string) – Local file path
  • $rempath (string) - caminho do arquivo remoto
  • $mode (string) – FTP mode, defaults to ‘auto’ (options are: ‘auto’, ‘binary’, ‘ascii’)
  • $permissions (int) – File permissions (octal)
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Uploads a file to your server. You must supply the local path and the caminho remoto, e você pode, opcionalmente, definir o modo e as permissões. Example:

$this->ftp->upload('/local/path/to/myfile.html', '/public_html/myfile.html', 'ascii', 0775);

If ‘auto’ mode is used it will base the mode on the file extension of the source file.

If set, permissions have to be passed as an octal value.

download($rempath, $locpath[, $mode = 'auto'])¶
Parâmetros:
  • $rempath (string) - caminho do arquivo remoto
  • $locpath (string) – Local file path
  • $mode (string) – FTP mode, defaults to ‘auto’ (options are: ‘auto’, ‘binary’, ‘ascii’)
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Faz o download de um arquivo do seu servidor. Você deve fornecer o caminho remoto e the local path, and you can optionally set the mode. Example:

$this->ftp->download('/public_html/myfile.html', '/local/path/to/myfile.html', 'ascii');

If ‘auto’ mode is used it will base the mode on the file extension of the source file.

Retorna FALSE se o download não for executado com sucesso (inclusive se o PHP não tiver permissão para gravar o arquivo local).

rename($old_file, $new_file[, $move = FALSE])¶
Parâmetros:
  • $old_file (string) - nome do arquivo antigo
  • $new_file (string) - Novo nome de arquivo
  • $move (bool) - Se um movimento está sendo realizado
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Permits you to rename a file. Supply the source file name/path and the new file name/path.

// Renames green.html to blue.html
$this->ftp->rename('/public_html/foo/green.html', '/public_html/foo/blue.html');
move($old_file, $new_file)¶
Parâmetros:
  • $old_file (string) - nome do arquivo antigo
  • $new_file (string) - Novo nome de arquivo
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Lets you move a file. Supply the source and destination paths:

// Moves blog.html from "joe" to "fred"
$this->ftp->move('/public_html/joe/blog.html', '/public_html/fred/blog.html');

Nota

If the destination file name is different the file will be renamed.

Verdadeiro sucesso, FALSE em falha

bool

$this->ftp->delete_file('/public_html/joe/blog.html');
delete_dir($filepath)¶
Parâmetros:
  • $filepath (string) – Path to directory to delete
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Lets you delete a directory and everything it contains. Supply the caminho de origem para o diretório com uma barra à direita.

Importante

Be VERY careful with this method! It will recursively delete everything within the supplied path, including sub-folders and all files. Make absolutely sure your path is correct. Try using list_files() first to verify that your path is correct.

$this->ftp->delete_dir('/public_html/path/to/folder/');
list_files([$path = '.'])¶
Parâmetros:
  • $path (string) – Directory path
Returns:

Uma lista de matriz de arquivos ou FALSE em falha
Return type:

array

Permits you to retrieve a list of files on your server returned as an array. Você deve fornecer o caminho para o diretório desejado.

$list = $this->ftp->list_files('/public_html/');
print_r($list);
mirror($locpath, $rempath)¶
Parâmetros:
  • $locpath (string) – Local path
  • $rempath (string) - caminho remoto
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Lê recursivamente uma pasta local e tudo o que ela contém (incluindo sub-folders) and creates a mirror via FTP based on it. Whatever the directory structure of the original file path will be recreated on the servidor. Você deve fornecer um caminho de origem e um caminho de destino:

$this->ftp->mirror('/path/to/myfolder/', '/public_html/myfolder/');
mkdir($path[, $permissions = NULL])¶
Parâmetros:
  • $path (string) - Caminho para o diretório para criar
  • $permissions (int) – Permissions (octal)
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Permite criar um diretório no seu servidor. Forneça o caminho que termina em the folder name you wish to create, with a trailing slash.

Permissions can be set by passing an octal value in the second parameter.

// Creates a folder named "bar"
$this->ftp->mkdir('/public_html/foo/bar/', 0755);
chmod($path, $perm)¶
Parâmetros:
  • $path (string) – Path to alter permissions for
  • $perm (int) – Permissions (octal)
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Permite que você defina permissões de arquivo. Forneça o caminho para o arquivo ou directory you wish to alter permissions on:

// Chmod "bar" to 755
$this->ftp->chmod('/public_html/foo/bar/', 0755);
changedir($path[, $suppress_debug = FALSE])¶
Parâmetros:

Returns:

Verdadeiro sucesso, FALSE em falha

bool

Altera o diretório de trabalho atual para o caminho especificado.

The $suppress_debug parameter is useful in case you want to use this method as an is_dir() alternative for FTP.

Working with the Classe FTP¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe FTP é inicializada em seu Controller usando a função $this-> load-> library:

$this->load->library('ftp');

Classe FTP¶

CodeIgniter’s Classe FTP permits files to be transferred to a remote servidor. Arquivos remotos também podem ser movidos, renomeados e excluídos. O FTP classe também inclui uma função de "espelhamento" que permite que um diretório a ser recriado remotamente via FTP.

Nota

Os protocolos SFTP e SSL FTP não são suportados, somente padrão FTP.

Parâmetros:
  • $path (string) - Caminho de diretório
Retorna:

Uma lista de matriz de arquivos ou FALSE em falha
Tipo de retorno:

array

Permite recuperar uma lista de arquivos em seu servidor retornados como array. Você deve fornecer o caminho para o diretório desejado.

$list = $this->ftp->list_files('/public_html/');

mirror($locpath, $rempath)¶
Parâmetros:
  • $locpath (string) - caminho local
  • $rempath (string) - caminho remoto
Retorna:

Verdadeiro sucesso, FALSE em falha
Tipo de retorno:

bool

Lê recursivamente uma pasta local e tudo o que ela contém (incluindo sub-pastas) e cria um espelho via FTP com base nele. Qualquer que seja estrutura do diretório do caminho do arquivo original será recriada no servidor. Você deve fornecer um caminho de origem e um caminho de destino:

$this->ftp->mirror('/path/to/myfolder/', '/public_html/myfolder/');
mkdir($path[, $permissions = NULL])¶
Parâmetros:
  • $path (string) - Caminho para o diretório para criar
  • $permissions (int) - Permissões (octal)
Retorna:

Verdadeiro sucesso, FALSE em falha
Tipo de retorno:

bool

Permite criar um diretório no seu servidor. Forneça o caminho que termina em o nome da pasta que você deseja criar, com uma barra à direita.

Permissões podem ser definidas passando um valor octal no segundo parâmetro.

// Creates a folder named "bar"
$this->ftp->mkdir('/public_html/foo/bar/', 0755);

  • $perm (int) - Permissões (octal)

// Chmod "bar" to 755

  • $suppress_debug (bool) - Se deseja desativar mensagens de depuração para este comando

bool

Altera o diretório de trabalho atual para o caminho especificado.

The $suppress_debug parameter is useful in case you want to use this method as an is_dir() alternative for FTP.
