Classe de agente do usuário¶

The Classe de agente do usuário provides functions that help identify information sobre o navegador, dispositivo móvel ou robô visitando seu site. Em addition you can get referrer information as well as language and supported character-set information.

  • Using the Classe de agente do usuário
    • Inicializando a classe
    • User Agent Definitions
    • Example
  • Class Reference

Using the Classe de agente do usuário¶

Inicializando a classe¶

Like most other classes in CodeIgniter, the User Agent class is initialized in your controller using the $this->load->library function:

$this->load->library('user_agent');

Uma vez carregado, o objeto estará disponível usando: $this->agent

User Agent Definitions¶

The user agent name definitions are located in a config file located at: application/config/user_agents.php. You may add items to the various user agent arrays if needed.

Example¶

When the User Agent class is initialized it will attempt to determine se o agente do usuário navegando em seu site é um navegador da Web, um device, or a robot. It will also gather the platform information if it está disponível.

$this->load->library('user_agent');

if ($this->agent->is_browser())
{
        $agent = $this->agent->browser().' '.$this->agent->version();
}
elseif ($this->agent->is_robot())
{
        $agent = $this->agent->robot();
}
elseif ($this->agent->is_mobile())
{
        $agent = $this->agent->mobile();
}
else
{
        $agent = 'Unidentified User Agent';
}

echo $agent;

echo $this->agent->platform(); // Platform info (Windows, Linux, Mac, etc.)

Class Reference¶

class CI_User_agent¶
is_browser([$key = NULL])¶
Parâmetros:
  • $key (string) - Nome do navegador opcional
Returns:

TRUE if the user agent is a (specified) browser, FALSE if not
Return type:

bool

Returns TRUE/FALSE (boolean) if the user agent is a known web browser.

if ($this->agent->is_browser('Safari'))
{
        echo 'You are using Safari.';
}
elseif ($this->agent->is_browser())
{
        echo 'You are using a browser.';
}

Nota

The string “Safari” in this example is an array key in the list of browser definitions. Você pode encontrar esta lista em application/config/user_agents.php se você quiser adicionar novos browsers or change the stings.

is_mobile([$key = NULL])¶
Parâmetros:
  • $key (string) – Optional mobile device name
Returns:

TRUE if the user agent is a (specified) mobile device, FALSE if not
Return type:

bool

Returns TRUE/FALSE (boolean) if the user agent is a known mobile device.

if ($this->agent->is_mobile('iphone'))
{
        $this->load->view('iphone/home');
}
elseif ($this->agent->is_mobile())
{
        $this->load->view('mobile/home');
}
else
{
        $this->load->view('web/home');
}
is_robot([$key = NULL])¶
Parâmetros:
  • $key (string) – Optional robot name
Returns:

TRUE if the user agent is a (specified) robot, FALSE if not
Return type:

bool

Retorna TRUE/FALSE (booleano) se o agente do usuário for um robô conhecido.

Nota

The user agent library only contains the most common robot definitions. It is not a complete list of bots. There are hundreds of them so searching for each one would not be very efficient. If you find that some bots that commonly visit your site are missing from the list you can add them to your application/config/user_agents.php file.

is_referral()¶
Returns:TRUE se o agente do usuário for uma referência, FALSE se não for
Return type:bool

Retorna TRUE/FALSE (booleano) se o agente do usuário foi encaminhado de outro site.

browser()¶
Returns:Navegador detectado ou uma string vazia
Return type:string

Returns a string containing the name of the web browser viewing your site.

version()¶
Returns:Detected browser version or an empty string
Return type:string

Returns a string containing the version number of the web browser viewing your site.

mobile()¶
Returns:Marca de dispositivo móvel detectado ou uma string vazia
Return type:string

Returns a string containing the name of the mobile device viewing your site.

robot()¶
Returns:Nome do robô detectado ou uma string vazia
Return type:string

Retorna uma string contendo o nome do robô visualizando seu site.

platform()¶
Returns:Detected operating system or an empty string
Return type:string

Returns a string containing the platform viewing your site (Linux, Windows, OS X, etc.).

referrer()¶
Returns:Referenciador detectado ou uma string vazia
Return type:string

O referenciador, se o agente do usuário foi encaminhado de outro site. Normalmente você vai testar para isso da seguinte forma:

if ($this->agent->is_referral())
{
        echo $this->agent->referrer();
}
agent_string()¶
Returns:Sequência completa do agente do usuário ou uma cadeia vazia
Return type:string

Retorna uma string contendo a string completa do agente do usuário. Normalmente, será algo assim:

Mozilla/5.0 (Macintosh; U; Intel Mac OS X; en-US; rv:1.8.0.4) Gecko/20060613 Camino/1.0.2
accept_lang([$lang = 'en'])¶
Parâmetros:
  • $lang (string) – Language key
Returns:

TRUE se a linguagem for aceita, FALSE se não for
Return type:

bool

Lets you determine if the user agent accepts a particular language. Example:

if ($this->agent->accept_lang('en'))
{
        echo 'You accept English!';
}

Nota

This method is not typically very reliable since some browsers do not provide language info, and even among those that do, it is not always accurate.

languages()¶
Returns:Uma lista de matriz de idiomas aceitos
Return type:array

Retorna uma matriz de idiomas suportados pelo agente do usuário.

accept_charset([$charset = 'utf-8'])¶
Parâmetros:
  • $charset (string) – Character set
Returns:

TRUE se o conjunto de caracteres for aceito, FALSE se não for
Return type:

bool

Lets you determine if the user agent accepts a particular character set. Example:

if ($this->agent->accept_charset('utf-8'))
{
        echo 'You browser supports UTF-8!';
}

Nota

This method is not typically very reliable since some browsers do not provide character-set info, and even among those that do, it is not always accurate.

charsets()¶
Returns:An array list of accepted character sets
Return type:array

Retorna uma matriz de conjuntos de caracteres aceitos pelo agente do usuário.

parse($string)¶
Parâmetros:
  • $string (string) – A custom user-agent string
Return type:

void

Parses a custom user-agent string, different from the one reported by the current visitor.

Quando a classe User Agent for inicializada, ela tentará determinar se o agente do usuário navegando em seu site é um navegador da Web, um dispositivo ou um robô. Ele também reunirá as informações da plataforma se está disponível.

$this->load->library('user_agent');

        $agent = 'Unidentified User Agent';

echo $agent;

Classe de agente do usuário¶

The Classe de agente do usuário provides functions that help identify information sobre o navegador, dispositivo móvel ou robô visitando seu site. Em Além disso, você pode obter informações de referência, bem como idioma e informações de conjunto de caracteres suportadas.

  • Using the Classe de agente do usuário
    • Inicializando a classe
    • User Agent Definitions
    • Exemplo
  • Referência de Classe

Using the Classe de agente do usuário¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe User Agent é inicializado em seu Controller usando a função $this-> load->library function:

$this->load->library('user_agent');

Uma vez carregado, o objeto estará disponível usando: $this->agent

User Agent Definitions¶

As definições de nome do agente do usuário estão localizadas em um arquivo de configuração localizado em: application/config/user_agents.php. Você pode adicionar itens aos vários arrays do agente do usuário, se necessário.

Exemplo¶

Quando a classe User Agent for inicializada, ela tentará determinar se o agente do usuário navegando em seu site é um navegador da Web, um dispositivo ou um robô. Ele também reunirá as informações da plataforma se está disponível.

$this->load->library('user_agent');

if ($this->agent->is_browser())
{
        $agent = $this->agent->browser().' '.$this->agent->version();
}
elseif ($this->agent->is_robot())
{
        $agent = $this->agent->robot();
}
elseif ($this->agent->is_mobile())
{
        $agent = $this->agent->mobile();
}
else
{
        $agent = 'Unidentified User Agent';
}

echo $agent;

echo $this->agent->platform(); // Platform info (Windows, Linux, Mac, etc.)

is_browser([$key = NULL])¶
Parâmetros:
  • $key (string) - Nome do navegador opcional
Retorna:

TRUE se o agente do usuário for um navegador (especificado), FALSE se não for
Tipo de retorno:

bool

Retorna TRUE/FALSE (booleano) se o agente do usuário for um navegador da Web conhecido.

if ($this->agent->is_browser('Safari'))
{
        echo 'You are using Safari.';
}
elseif ($this->agent->is_browser())
{
        echo 'You are using a browser.';
}

Nota

A string "Safari" neste exemplo é uma chave de matriz na lista de definições do navegador. Você pode encontrar esta lista em application/config/user_agents.php se você quiser adicionar novos navegadores ou altere as picadas.

bool
