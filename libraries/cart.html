Classee de carrinho de compras¶

The Cart Classe permits items to be added to a session that stays active while a user is browsing your site. These items can be retrieved and displayed in a standard “shopping cart” format, allowing the user to atualizar a quantidade ou remover itens do carrinho.

Importante

The Cart library is DEPRECATED and should not be used. It is currently only kept for backwards compatibility.

Please note that the Cart Classe ONLY provides the core “cart” functionality. It does not provide shipping, credit card authorization, or other processing components.

Usando a classe Cart¶

Initializing the Classee de carrinho de compras¶

Importante

A classe Cart utiliza o CodeIgniter's Session Classe to save the cart information to a database, so antes de usar a classe Cart você deve configurar uma tabela de database como indicated in the Session Documentation, and set the session preferences in your application/config/config.php file to utilize a database.

To initialize the Classee de carrinho de compras in your controller constructor, use the $this->load->library() method:

$this->load->library('cart');

Once loaded, the Cart object will be available using:

$this->cart

Nota

The Cart Classe will load and initialize the Session Classe automaticamente, a menos que você esteja usando sessões em outro lugar aplicativo, você não precisa carregar a classe Session.

Adding an Item to The Cart¶

Para adicionar um item ao carrinho de compras, simplesmente passe um array com o product information to the $this->cart->insert() method, as shown below:

$data = array(
        'id'      => 'sku_123ABC',
        'qty'     => 1,
        'price'   => 39.95,
        'name'    => 'T-Shirt',
        'options' => array('Size' => 'L', 'Color' => 'Red')
);

$this->cart->insert($data);

Importante

The first four array indexes above (id, qty, price, and nome) são required. If you omit any of them the data will not be saved to the cart. The fifth index (options) is optional. It is intended para ser usado nos casos em que seu produto tem opções associadas a ele. Use an array for options, as shown above.

The five reserved indexes are:

  • id - Each product in your store must have a unique identifier. Normalmente, isso será um "sku" ou outro identificador desse tipo.
  • qty - The quantity being purchased.
  • price - O preço do item.
  • name - The name of the item.
  • options - Any additional attributes that are needed to identify the product. These must be passed via an array.

Além dos cinco índices acima, existem duas palavras reservadas: rowid and subtotal. These are used internally by the Cart class, so please do NOT use those words as index names when inserting data into the cart.

Sua matriz pode conter dados adicionais. Qualquer coisa que você inclua no seu array will be stored in the session. However, it is best to standardize seus dados entre todos os seus produtos, a fim de tornar a exibição do informações em uma tabela mais fácil.

$data = array(
        'id'      => 'sku_123ABC',
        'qty'     => 1,
        'price'   => 39.95,
        'name'    => 'T-Shirt',
        'coupon'         => 'XMAS-50OFF'
);

$this->cart->insert($data);

The insert() method will return the $rowid if you successfully insert a single item.

Adicionando vários itens ao carrinho¶

Usando uma matriz multidimensional, como mostrado abaixo, é possível adicione vários produtos ao carrinho em uma ação. Isso é útil em casos onde você deseja permitir que as pessoas selecionem entre vários itens no same page.

$data = array(
        array(
                'id'      => 'sku_123ABC',
                'qty'     => 1,
                'price'   => 39.95,
                'name'    => 'T-Shirt',
                'options' => array('Size' => 'L', 'Color' => 'Red')
        ),
        array(
                'id'      => 'sku_567ZYX',
                'qty'     => 1,
                'price'   => 9.95,
                'name'    => 'Coffee Mug'
        ),
        array(
                'id'      => 'sku_965QRS',
                'qty'     => 1,
                'price'   => 29.95,
                'name'    => 'Shot Glass'
        )
);

$this->cart->insert($data);

Displaying the Cart¶

Para exibir o carrinho, você criará um view file com código semelhante ao mostrado abaixo.

Por favor, note que este exemplo usa o form helper.

<?php echo form_open('path/to/controller/update/method'); ?>

<table cellpadding="6" cellspacing="1" style="width:100%" border="0">

<tr>
        <th>QTY</th>
        <th>Item Description</th>
        <th style="text-align:right">Item Price</th>
        <th style="text-align:right">Sub-Total</th>
</tr>

<?php $i = 1; ?>

<?php foreach ($this->cart->contents() as $items): ?>

        <?php echo form_hidden($i.'[rowid]', $items['rowid']); ?>

        <tr>
                <td><?php echo form_input(array('name' => $i.'[qty]', 'value' => $items['qty'], 'maxlength' => '3', 'size' => '5')); ?></td>
                <td>
                        <?php echo $items['name']; ?>

                        <?php if ($this->cart->has_options($items['rowid']) == TRUE): ?>

                                <p>
                                        <?php foreach ($this->cart->product_options($items['rowid']) as $option_name => $option_value): ?>

                                                <strong><?php echo $option_name; ?>:</strong> <?php echo $option_value; ?><br />

                                        <?php endforeach; ?>
                                </p>

                        <?php endif; ?>

                </td>
                <td style="text-align:right"><?php echo $this->cart->format_number($items['price']); ?></td>
                <td style="text-align:right">$<?php echo $this->cart->format_number($items['subtotal']); ?></td>
        </tr>

<?php $i++; ?>

<?php endforeach; ?>

<tr>
        <td colspan="2"> </td>
        <td class="right"><strong>Total</strong></td>
        <td class="right">$<?php echo $this->cart->format_number($this->cart->total()); ?></td>
</tr>

</table>

<p><?php echo form_submit('', 'Update your Cart'); ?></p>

Updating The Cart¶

To update the information in your cart, you must pass an array contendo o ID da Linha e uma ou mais propriedades pré-definidas para o $this->cart->update() method.

Nota

If the quantity is set to zero, the item will be removed from the cart.

$data = array(
        'rowid' => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'   => 3
);

$this->cart->update($data);

// Or a multi-dimensional array

$data = array(
        array(
                'rowid'   => 'b99ccdf16028f015540f341130b6d8ec',
                'qty'     => 3
        ),
        array(
                'rowid'   => 'xw82g9q3r495893iajdh473990rikw23',
                'qty'     => 4
        ),
        array(
                'rowid'   => 'fh4kdkkkaoe30njgoe92rkdkkobec333',
                'qty'     => 2
        )
);

$this->cart->update($data);

You may also update any property you have previously defined when inserting the item such as options, price or other custom fields.

$data = array(
        'rowid'  => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'    => 1,
        'price'  => 49.95,
        'coupon' => NULL
);

$this->cart->update($data);

What is a Row ID?¶

The row ID is a unique identifier that is generated by the cart code when an item is added to the cart. The reason a unique ID is created is so that identical products with different options can be managed by the cart.

Por exemplo, digamos que alguém compre duas camisetas idênticas (mesmo produto ID), but in different sizes. The product ID (and other attributes) will be identical for both sizes because it’s the same shirt. The only diferença será o tamanho. O carrinho deve, portanto, ter um meio de identificando esta diferença para que os dois tamanhos de camisas possam ser managed independently. It does so by creating a unique “row ID” based on the product ID and any options associated with it.

In nearly all cases, updating the cart will be something the user does via the “view cart” page, so as a developer, it is unlikely that you will ever have to concern yourself with the “row ID”, other than making sure your “view cart” page contains this information in a hidden form field, and making sure it gets passed to the update() method when the update form is submitted. Please examine the construction of the “view cart” page above for more information.

Estas são as regras de expressões regulares que usamos para validar o produto ID - alpha-numeric, dashes, underscores, or periods by default

Estas são as regras de expressões regulares que usamos para validar o produto ID and product name - alpha-numeric, dashes, underscores, colons or periods by default

$product_name_safe = TRUE

Whether or not to only allow safe product names. Default TRUE.

insert([$items = array()])¶
Parâmetros:
  • $items (array) – Items to insert into the cart
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Insira itens no carrinho e salve-os na tabela de sessões. Retorna TRUE on success and FALSE on failure.

update([$items = array()])¶
Parâmetros:
  • $items (array) - Itens para atualizar no carrinho
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

This method permits changing the properties of a given item. Typically it is called from the “view cart” page if a user makes changes to the quantity before checkout. That array must contain the rowid para cada item.

remove($rowid)¶
Parâmetros:
  • $rowid (int) – ID of the item to remove from the cart
Returns:

Verdadeiro sucesso, FALSE em falha
Return type:

bool

Permite que você remova um item do carrinho de compras passando-o $rowid.

total()¶
Returns:Total amount
Return type:int

Exibe o valor total no carrinho.

total_items()¶
Returns:Total amount of items in the cart
Return type:int

Displays the total number of items in the cart.

contents([$newest_first = FALSE])¶
Parâmetros:
  • $newest_first (bool) - Se deseja ordenar o array com os itens mais novos primeiro
Returns:

An array of cart contents
Return type:

array

Retorna um array contendo tudo no carrinho. Você pode classificar o ordem pela qual o array é retornado passando-se TRUE onde o conteúdo será classificado do mais recente para o mais antigo, caso contrário, é classificado de mais antigo to newest.

get_item($row_id)¶
Parâmetros:
  • $row_id (int) – Row ID to retrieve
Returns:

Array of item data
Return type:

array

Returns an array containing data for the item matching the specified row ID, or FALSE if no such item exists.

has_options($row_id = '')¶
Parâmetros:
  • $row_id (int) – Row ID to inspect
Returns:

TRUE if options exist, FALSE otherwise
Return type:

bool

Returns TRUE (boolean) if a particular row in the cart contains options. This method is designed to be used in a loop with contents(), since you must pass the rowid to this method, as shown in the Displaying o exemplo do carrinho acima.

product_options([$row_id = ''])¶
Parâmetros:
  • $row_id (int) – Row ID
Returns:

Matriz de opções de produtos
Return type:

array

Retorna uma matriz de opções para um produto específico. Este método é designed to be used in a loop with contents(), since you must pass the rowid to this method, as shown in the Displaying the Exemplo de carrinho acima.

destroy()¶
Return type:void

Permits you to destroy the cart. This method will likely be called quando terminar de processar o pedido do cliente.

Classe de carrinho de compras¶

A classe Cart permite que itens sejam adicionados a uma sessão que permanece ativa enquanto um usuário está navegando em seu site. Esses itens podem ser recuperados e exibido em um formato padrão de "carrinho de compras", permitindo ao usuário atualizar a quantidade ou remover itens do carrinho.

Importante

A Library de carrinho é DEPRECADA e não deve ser usada. Atualmente, ele é mantido apenas para compatibilidade com versões anteriores.

Por favor, note que a classe de carrinho oferece apenas o núcleo "carrinho" funcionalidade. Não fornece frete, autorização de cartão de crédito, ou outros componentes de processamento.

Usando a classe Cart¶

Initializing the Classe de carrinho de compras¶

Importante

A classe Cart utiliza o CodeIgniter's Sessão Class para salvar as informações do carrinho em um Database, antes de usar a classe Cart você deve configurar uma tabela de database como indicado no Sessão Documentatione defina preferências de sessão em seu arquivo application/config/config.php para utilizar um Database.

To initialize the Classe de carrinho de compras in your controller constructor, use the $this->load->library() method:

$this->load->library('cart');

Uma vez carregado, o objeto Cart estará disponível usando:

$this->cart

Nota

The Cart Class will load and initialize the Sessão Class automaticamente, a menos que você esteja usando sessões em outro lugar application, you do not need to load the Sessão class.

Adicionando um item ao carrinho¶

Para adicionar um item ao carrinho de compras, simplesmente passe um array com o product information to the $this->cart->insert() method, as shown below:

$data = array(
        'id'      => 'sku_123ABC',
        'qty'     => 1,
        'price'   => 39.95,
        'name'    => 'T-Shirt',
        'options' => array('Size' => 'L', 'Color' => 'Red')
);

$this->cart->insert($data);

Importante

Os primeiros quatro índices de array acima (id, qty, price e nome) são required. Se você omitir algum deles, os dados não serão salvo no carrinho. O quinto índice (opções) é opcional. Destina-se para ser usado nos casos em que seu produto tem opções associadas a ele. Use uma matriz para opções, como mostrado acima.

Os cinco índices reservados são:

  • id - Cada produto em sua loja deve ter um identificador exclusivo. Normalmente, isso será um "sku" ou outro identificador desse tipo.
  • qty - A quantidade a ser comprada.
  • price - O preço do item.
  • name - O nome do objeto.
  • options - Quaisquer atributos adicionais necessários para identificar o produto. Estes devem ser passados através de um array.

Além dos cinco índices acima, existem duas palavras reservadas: rowid e subtotal. Estes são usados internamente pela classe Cart, então por favor, não use essas palavras como nomes de índice ao inserir dados em o carrinho.

Sua matriz pode conter dados adicionais. Qualquer coisa que você inclua no seu array será armazenado na sessão. No entanto, é melhor padronizar seus dados entre todos os seus produtos, a fim de tornar a exibição do informações em uma tabela mais fácil.

$data = array(
        'id'      => 'sku_123ABC',
        'qty'     => 1,
        'price'   => 39.95,
        'name'    => 'T-Shirt',
        'coupon'         => 'XMAS-50OFF'
);

$this->cart->insert($data);

The insert() method will return the $rowid if you successfully insert a único item.

Adicionando vários itens ao carrinho¶

Usando uma matriz multidimensional, como mostrado abaixo, é possível adicione vários produtos ao carrinho em uma ação. Isso é útil em casos onde você deseja permitir que as pessoas selecionem entre vários itens no mesma página.

$data = array(
        array(
                'id'      => 'sku_123ABC',
                'qty'     => 1,
                'price'   => 39.95,
                'name'    => 'T-Shirt',
                'options' => array('Size' => 'L', 'Color' => 'Red')
        ),
        array(
                'id'      => 'sku_567ZYX',
                'qty'     => 1,
                'price'   => 9.95,
                'name'    => 'Coffee Mug'
        ),
        array(
                'id'      => 'sku_965QRS',
                'qty'     => 1,
                'price'   => 29.95,
                'name'    => 'Shot Glass'
        )
);

$this->cart->insert($data);

Exibindo o carrinho¶

Para exibir o carrinho, você criará um view file com código semelhante ao mostrado abaixo.

Por favor, note que este exemplo usa o form helper.

<?php echo form_open('path/to/controller/update/method'); ?>

Atualizando o carrinho¶

Para atualizar as informações no seu carrinho, você deve passar um array contendo o ID da Linha e uma ou mais propriedades pré-definidas para o $this->cart->update() method.

Nota

Se a quantidade for definida como zero, o item será removido de o carrinho.

$data = array(
        'rowid' => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'   => 3
);

$this->cart->update($data);

// Or a multi-dimensional array

$data = array(
        array(
                'rowid'   => 'b99ccdf16028f015540f341130b6d8ec',
                'qty'     => 3
        ),
        array(
                'rowid'   => 'xw82g9q3r495893iajdh473990rikw23',
                'qty'     => 4
        ),
        array(
                'rowid'   => 'fh4kdkkkaoe30njgoe92rkdkkobec333',
                'qty'     => 2
        )
);

$this->cart->update($data);

Você também pode atualizar qualquer propriedade que tenha definido anteriormente quando inserindo o item, como opções, preço ou outros campos personalizados.

$data = array(
        'rowid'  => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'    => 1,
        'price'  => 49.95,
        'coupon' => NULL
);

$this->cart->update($data);

O que é uma ID de linha?¶

O ID da linha é um identificador exclusivo gerado pelo código do carrinho when an item is added to o carrinho. The reason a unique ID is created é para que produtos idênticos com diferentes opções possam ser gerenciados by o carrinho.

Por exemplo, digamos que alguém compre duas camisetas idênticas (mesmo produto ID), mas em tamanhos diferentes. O ID do produto (e outros atributos) Seja idêntico para ambos os tamanhos porque é a mesma camisa. O único diferença será o tamanho. O carrinho deve, portanto, ter um meio de identificando esta diferença para que os dois tamanhos de camisas possam ser gerenciado de forma independente. Isso é feito criando um "ID de linha" exclusivo baseado o ID do produto e quaisquer opções associadas a ele.

Em quase todos os casos, atualizar o carrinho é algo que o usuário faz através da página "ver carrinho", por isso, como um desenvolvedor, é improvável que você alguma vez terá que se preocupar com o "ID da fileira", além de fazer Certifique-se de que sua página "carrinho de exibição" contenha essas informações em um formato oculto field, and making sure it gets passed to the update() method when o formulário de atualização é enviado. Por favor, examine a construção do Página "ver carrinho" acima para mais informações.

$this->cart->insert($data);

The insert() method will return the $rowid if you successfully insert a único item.

Atualizando o carrinho¶

Para atualizar as informações no seu carrinho, você deve passar um array contendo o ID da Linha e uma ou mais propriedades pré-definidas para o $this->cart->update() method.

Nota

Se a quantidade for definida como zero, o item será removido de the cart.

$data = array(
        'rowid' => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'   => 3
);

$this->cart->update($data);

// Or a multi-dimensional array

$data = array(
        array(
                'rowid'   => 'b99ccdf16028f015540f341130b6d8ec',
                'qty'     => 3
        ),
        array(
                'rowid'   => 'xw82g9q3r495893iajdh473990rikw23',
                'qty'     => 4
        ),
        array(
                'rowid'   => 'fh4kdkkkaoe30njgoe92rkdkkobec333',
                'qty'     => 2
        )
);

$this->cart->update($data);

Você também pode atualizar qualquer propriedade que tenha definido anteriormente quando inserindo o item, como opções, preço ou outros campos personalizados.

$data = array(
        'rowid'  => 'b99ccdf16028f015540f341130b6d8ec',
        'qty'    => 1,
        'price'  => 49.95,
        'coupon' => NULL
);

$this->cart->update($data);

O que é uma ID de linha?¶

O ID da linha é um identificador exclusivo gerado pelo código do carrinho quando um item é adicionado ao carrinho. A razão pela qual um ID único é criado é para que produtos idênticos com diferentes opções possam ser gerenciados pelo carrinho.

Por exemplo, digamos que alguém compre duas camisetas idênticas (mesmo produto ID), mas em tamanhos diferentes. O ID do produto (e outros atributos) Seja idêntico para ambos os tamanhos porque é a mesma camisa. O único diferença será o tamanho. O carrinho deve, portanto, ter um meio de identificando esta diferença para que os dois tamanhos de camisas possam ser gerenciado de forma independente. Isso é feito criando um "ID de linha" exclusivo baseado o ID do produto e quaisquer opções associadas a ele.

Em quase todos os casos, atualizar o carrinho é algo que o usuário faz através da página "ver carrinho", por isso, como um desenvolvedor, é improvável que você alguma vez terá que se preocupar com o "ID da fileira", além de fazer Certifique-se de que sua página "carrinho de exibição" contenha essas informações em um formato oculto field, and making sure it gets passed to the update() method when o formulário de atualização é enviado. Por favor, examine a construção do Página "ver carrinho" acima para mais informações.
