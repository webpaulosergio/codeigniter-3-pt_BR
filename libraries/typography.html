Tipografia de Classe¶

The Tipografia de Classe provides methods that help you format text.

  • Using the Tipografia de Classe
    • Inicializando a classe
  • Class Reference

Using the Tipografia de Classe¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe Typography é initialized in your controller using the $this->load->library() method:

$this->load->library('typography');

Uma vez carregado, o objeto da library Typography estará disponível usando:

$this->typography

Class Reference¶

class CI_Typography¶
$protect_braced_quotes = FALSE

Ao usar a library Typography em conjunto com o Template Parser library it can often be desirable to protect single and double quotes within curly braces. To enable this, set the protect_braced_quotes class property to TRUE.

Exemplo de uso:

$this->load->library('typography');
$this->typography->protect_braced_quotes = TRUE;
auto_typography($str[, $reduce_linebreaks = FALSE])¶
Parâmetros:
  • $str (string) – Input string
  • $reduce_linebreaks (bool) – Whether to reduce consequitive linebreaks
Returns:

HTML typography-safe string
Return type:

string

Formats text so that it is semantically and typographically correct HTML. Takes a string as input and returns it with the following formatting:

  • Surrounds paragraphs within <p></p> (looks for double line breaks to identify paragraphs).
  • Single line breaks are converted to <br />, except those that appear within <pre> tags.
  • Block level elements, like <div> tags, are not wrapped within paragraphs, but their contained text is if it contains paragraphs.
  • As cotações são convertidas em entidades de cotação curvas corretamente, exceto aquelas que aparecem em tags.
  • Apostrophes are converted to curly apostrophe entities.
  • Double dashes (either like – this or like–this) are converted to em—dashes.
  • Three consecutive periods either preceding or following a word are converted to ellipsis (…).
  • Double spaces following sentences are converted to non-breaking spaces to mimic double spacing.

Exemplo de uso:

$string = $this->typography->auto_typography($string);

Existe um parâmetro opcional que determina se o analisador deve reduzir mais de duas quebras de linha consecutivas para duas. Pass boolean TRUE to enable reducing line breaks:

$string = $this->typography->auto_typography($string, TRUE);

Nota

A formatação tipográfica pode exigir muito do processador, principalmente se você tiver muito conteúdo sendo formatado. If you choose to use this method you may want to consider caching suas páginas.

format_characters($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

Cadeia formatada
Return type:

string

This method is similar to auto_typography() above, except that it only does character conversion:

  • As cotações são convertidas em entidades de cotação curvas corretamente, exceto aquelas que aparecem em tags.
  • Apostrophes are converted to curly apostrophe entities.
  • Double dashes (either like – this or like–this) are converted to em—dashes.
  • Three consecutive periods either preceding or following a word are converted to ellipsis (…).
  • Double spaces following sentences are converted to non-breaking spaces to mimic double spacing.

Exemplo de uso:

$string = $this->typography->format_characters($string);
nl2br_except_pre($str)¶
Parâmetros:
  • $str (string) – Input string
Returns:

Cadeia formatada
Return type:

string

Converts newlines to <br /> tags unless they appear within <pre> tags. This method is identical to the native PHP nl2br() function, except that it ignores <pre> tags.

Exemplo de uso:

$string = $this->typography->nl2br_except_pre($string);

Classe de Tipografia¶

The Classe de Tipografia provides methods that help you format text.

  • Using the Classe de Tipografia
    • Inicializando a classe
  • Referência de Classe

Using the Classe de Tipografia¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe Typography é initialized in your controller using the $this->load->library() method:

$this->load->library('typography');

Uma vez carregado, o objeto da library Typography estará disponível usando:

$this->typography

Referência de Classe¶

class CI_Typography¶
$protect_braced_quotes = FALSE

Ao usar a library Typography em conjunto com o Library do analisador de Model muitas vezes, pode ser desejável proteger as aspas simples e duplas entre chaves. To enable this, set the protect_braced_quotes class property to TRUE.

Exemplo de uso:

$this->load->library('typography');
$this->typography->protect_braced_quotes = TRUE;
auto_typography($str[, $reduce_linebreaks = FALSE])¶
Parâmetros:
  • $str (string) - string de entrada
  • $reduce_linebreaks (bool) - Se deve reduzir as quebras de linha consequentes
Retorna:

Sequência de tipografia segura para HTML
Tipo de retorno:

string

Formata o texto para que seja HTML semanticamente e tipograficamente correto. Toma uma string como entrada e a retorna com a seguinte formatação:

  • Envolve os parágrafos dentro de <p> </ p> (procura por quebras de linha duplas para identificar parágrafos).
  • Quebras de linha únicas são convertidas em <br />, exceto aquelas que aparecem dentro de tags <pre>.
  • Os elementos de nível de bloco, como as tags <div>, não são agrupados em parágrafos, mas seu texto contido é se contiver parágrafos.
  • As cotações são convertidas em entidades de cotação curvas corretamente, exceto aquelas que aparecem em tags.
  • Apóstrofos são convertidos em entidades de apóstrofo de curvas.
  • Traços duplos (como - isto ou algo assim) são convertidos em traços.
  • Três períodos consecutivos, anteriores ou posteriores a uma palavra, são convertidos em reticências ().
  • Espaços duplos após as sentenças são convertidos em espaços não separáveis para imitar o espaçamento duplo.

Exemplo de uso:

$string = $this->typography->auto_typography($string);

Existe um parâmetro opcional que determina se o analisador deve reduzir mais de duas quebras de linha consecutivas para duas. Passe o booleano TRUE para permitir a redução de quebras de linha:

$string = $this->typography->auto_typography($string, TRUE);

Nota

A formatação tipográfica pode exigir muito do processador, principalmente se você tiver muito conteúdo sendo formatado. Se você escolher usar esse método, talvez queira considerar caching suas páginas.

format_characters($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Cadeia formatada
Tipo de retorno:

string

This method is similar to auto_typography() above, except that it only does character conversion:

  • As cotações são convertidas em entidades de cotação curvas corretamente, exceto aquelas que aparecem em tags.
  • Apóstrofos são convertidos em entidades de apóstrofo de curvas.
  • Traços duplos (como - isto ou algo assim) são convertidos em traços.
  • Três períodos consecutivos, anteriores ou posteriores a uma palavra, são convertidos em reticências ().
  • Espaços duplos após as sentenças são convertidos em espaços não separáveis para imitar o espaçamento duplo.

Exemplo de uso:

$string = $this->typography->format_characters($string);
nl2br_except_pre($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Cadeia formatada
Tipo de retorno:

string

Converte novas linhas em tags <br />, a menos que apareçam dentro de <pre> tags. This method is identical to the native PHP nl2br() function, except that it ignores <pre> tags.

Exemplo de uso:

$string = $this->typography->nl2br_except_pre($string);

Classe de Tipografia¶

The Classe de Tipografia provides methods that help you format text.

  • Using the Classe de Tipografia
    • Inicializando a classe
  • Referência de Classe

Using the Classe de Tipografia¶

Inicializando a classe¶

Como a maioria das outras classes no CodeIgniter, a classe Typography é initialized in your controller using the $this->load->library() method:

$this->load->library('typography');

Uma vez carregado, o objeto da library Typography estará disponível usando:

$this->typography

Referência de Classe¶

class CI_Typography¶
$protect_braced_quotes = FALSE

Ao usar a library Typography em conjunto com o Library do analisador de Model muitas vezes, pode ser desejável proteger as aspas simples e duplas entre chaves. To enable this, set the protect_braced_quotes class property to TRUE.

Exemplo de uso:

$this->load->library('typography');
$this->typography->protect_braced_quotes = TRUE;
auto_typography($str[, $reduce_linebreaks = FALSE])¶
Parâmetros:
  • $str (string) - string de entrada
  • $reduce_linebreaks (bool) - Se deve reduzir as quebras de linha consequentes
Retorna:

Sequência de tipografia segura para HTML
Tipo de retorno:

string

Formata o texto para que seja HTML semanticamente e tipograficamente correto. Toma uma string como entrada e a retorna com a seguinte formatação:

  • Envolve os parágrafos dentro de <p> </ p> (procura por quebras de linha duplas para identificar parágrafos).
  • Quebras de linha únicas são convertidas em <br />, exceto aquelas que aparecem dentro de tags <pre>.
  • Os elementos de nível de bloco, como as tags <div>, não são agrupados em parágrafos, mas seu texto contido é se contiver parágrafos.
  • As cotações são convertidas em entidades de cotação curvas corretamente, exceto aquelas que aparecem em tags.
  • Apóstrofos são convertidos em entidades de apóstrofo de curvas.
  • Traços duplos (como - isto ou algo assim) são convertidos em traços.
  • Três períodos consecutivos, anteriores ou posteriores a uma palavra, são convertidos em reticências ().
  • Espaços duplos após as sentenças são convertidos em espaços não separáveis para imitar o espaçamento duplo.

Exemplo de uso:

$string = $this->typography->auto_typography($string);

Existe um parâmetro opcional que determina se o analisador deve reduzir mais de duas quebras de linha consecutivas para duas. Passe o booleano TRUE para permitir a redução de quebras de linha:

$string = $this->typography->auto_typography($string, TRUE);

Nota

A formatação tipográfica pode exigir muito do processador, principalmente se você tiver muito conteúdo sendo formatado. Se você escolher usar esse método, talvez queira considerar caching suas páginas.

format_characters($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Cadeia formatada
Tipo de retorno:

string

This method is similar to auto_typography() above, except that it only does character conversion:

  • As cotações são convertidas em entidades de cotação curvas corretamente, exceto aquelas que aparecem em tags.
  • Apóstrofos são convertidos em entidades de apóstrofo de curvas.
  • Traços duplos (como - isto ou algo assim) são convertidos em traços.
  • Três períodos consecutivos, anteriores ou posteriores a uma palavra, são convertidos em reticências ().
  • Espaços duplos após as sentenças são convertidos em espaços não separáveis para imitar o espaçamento duplo.

Exemplo de uso:

$string = $this->typography->format_characters($string);
nl2br_except_pre($str)¶
Parâmetros:
  • $str (string) - string de entrada
Retorna:

Cadeia formatada
Tipo de retorno:

string

Converte novas linhas em tags <br />, a menos que apareçam dentro de <pre> tags. This method is identical to the native PHP nl2br() function, except that it ignores <pre> tags.

Exemplo de uso:

$string = $this->typography->nl2br_except_pre($string);
